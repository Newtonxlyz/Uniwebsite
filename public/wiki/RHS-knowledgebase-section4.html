座椅

分类及组成

现代汽车座椅对于其设计形式有一些基本要求，对于前排座椅必须要有头枕，可以选择座椅是否具备扶手结 构。一般有前后、高度、角度调节，后排座椅可以前后调节，翻折或旋转。 按照座椅结构类型，我们一般将座椅分成以下几种形式:

  1. 折叠座椅。坐垫可以是整体或是分开的，靠背可以折叠，这种座椅一般用于后排以 及前排副驾驶座椅，用这个功能来增大后排行李箱的空间，对于大而长的行李可以 放入车中。
  2. 独立座椅。区别于其他座椅，独立座椅可以使每个乘员独立的调节座椅， 通常坐垫和靠背都是相对独立的，可以前后、翻转等调节，而不影响其他乘员。独立座椅可以是 40/60 或 50/50，根据车型风格定义。
  3. 长条座椅。长条座椅的坐垫为整体，靠背是可以分离或整体，分离式的靠背可以独 立折叠方便后排乘员的进出。
  4. 斗式座椅。这种座椅仅供单人使用，直接固定在车身上，有的可以旋转。

汽车座椅主要由骨架、座垫、靠背及其调节装置组成

  • 靠背骨架
    座椅靠背骨架一般有两种结构：PIPE 结构和 CHANNEL 结构。
    PIPE 结构的靠背是釆用钢管折弯焊接而成，这种结构简单，成本低，用于功能要求不高 的轿车。
    CHANNEL 结构一般是釆用钣金冲压焊接而成， 这种结构工艺复杂且成本高，但 这种座椅结构强度高，制造精度高，应用于功能复杂的座椅靠背骨架。
    靠背骨架强度对后碰乘员的挥鞭保护非常重要，成正相关关系，即强度越高，挥鞭保护越好。 对于 NCAP 中挥鞭保护有更高要的车辆，建议使用 CHANNEL靠背骨架。 在上世纪六七十年代，汽车发生追尾碰撞时，虽然追尾碰撞发生的概率较小，但造成 的伤害比例，以及医疗成本都比较高。一般来说，车辆被追尾的时候，相对速度都不高， 属于较为轻微的碰撞。因此倾向于将座椅靠背的刚度强度设计的比较弱，这样也可以减小 碰撞中，乘员身体所受到的载荷。后来对交通事故数据进行评估， 发现在一些比较严重的追尾事故中，例如小轿车被大货车追尾了，或者后车完全没有减速，乘员的伤亡比较惨 烈。通过对座椅的技术分析，发现座椅靠背刚度强度太弱，在剧烈的追尾碰撞下，座椅靠 背被乘员向后运动的冲击力加载，向后转动到与竖直方向的夹角超过60°之后，支撑不住 乘员了。乘员将从座椅靠背滑出去，后果可想而知。 所以后来将座椅靠背骨架做强，但靠背骨架太强，在低速碰撞下，乘员受到来自椅背 的作用力更剧烈， 乘员躯干向后运动的趋势被“硬”椅背限制住，因此加剧头部和躯干之间的相对运动，增加颈部受到挥鞭伤的风险。座椅靠背太强虽然照顾了高速工况的保护需 求，但牺牲了低速工况的保护效果。 后来，通用汽车发明了一种软硬兼有的座椅，即将座椅靠背外围骨架做的很强，而中 间留下一个空洞，用弹簧。这样骨架是保证在高速追尾碰撞的时候，座椅能承受较大的载 荷，把乘员给支撑住；而中间的弹簧则允许低速碰撞的时候，乘员背部陷入其中。此时乘员只受到挤压靠背泡沬和弹簧结构时所产生的反力，因此比较柔和，可减少颈部的挥鞭伤。这种座椅能使追尾碰撞事故中的致命伤害减少 46.3%目前的大部分汽车座椅依然采用这一设计思路。
  • 坐垫骨架
    坐垫骨架用于支撑整个坐垫，它对乘员保护性能影响非常大。坐垫骨架种类一般有三种: 全座盆形式、半座盆形式和弹簧形式。 全座盆骨架有足够的强度，在碰撞过程中，座盆不易变形，能很好的防止乘员下潜，对乘员保护非常有利。半座盆是在坐垫前端采用冲压钣金，用来提高强度，因为在碰撞中，坐 垫前端坡度上端是受力最大的地方，保证这里强度，对乘员保护有利。而弹簧的坐垫骨架是 用弹簧钢丝或硬钢丝加工而成，另外也有把用橡胶制作的弹性元件叫做座椅弹簧，座椅弹簧的特性决定了座椅的静、动态弹性特性。强度不足,碰撞中很容易塌陷，会给乘员带来伤害。 对于安全星级较高的车辆，座盆至少要选择半座盆。至少能保证坐垫前端的强度。
  • 滑轨
    滑轨是座椅的重要组成部分，通过它可以实现座椅的前后调节功能，同时也通过它将座 椅与整车底部连接起来。滑轨的主要功能有:
    1. 连接功能，将座椅与车身连接起来
    2. 运动功能，实现座椅相对车身的运动
    3. 固定功能，固定座椅，是座椅相对车身静止
    从安全角度，对滑轨的强度要求较高，滑轨的强度是座椅对乘员保护的根本，如果滑轨强度不足，则碰撞中变形会大，导致座椅失稳，严重影响乘员保护性能。在 NCAP 法规中有关于此的修正。例如，在 CNCAP 中，对前碰和挥鞭评估都有座椅滑轨强度的修正项。 考虑一定的安全余量， 在前期的设计和 CAE 分析中， 必须确保座椅的滑轨不能出现塑 性变形，不仅仅是滑轨本身，与滑轨连接的车身部分，也不能出现明显的塑性变形。 在NCAP 前碰工况中，对于固定滑轨的固定点处，轴向力能达到 12KN, 剪切力能达到 10kN因此前期对滑轨固定点处的强度要进行分析验证。
  • 头枕
    头枕的主要作用是用在后碰中，保护乘员免受挥鞭伤害。当后碰发生时，通过头枕将乘员的头托住，减小乘员的头部相对于胸部的运动幅度。 随着技术的发展，现在头枕已经从最初的固定式头枕，发展到现在的电子式主动头枕。 就是当后碰发生时，主动式头枕会被触发，向乘员后脑勺运动，消除乘员头部与头枕间的间 隙， 使头枕第一时间托住乘员头部，电子式主动头枕的核心技术在座椅供应商，作为主机厂进行挥鞭伤开发时，只要给供应商提出明 确的挥鞭保护要求即可，比如，满足哪些法规要求，至少能拿多少分等。因为座椅的挥鞭保 护性能，是座椅总成的单纯的一个性能，与其他约束系统无明显相关性。
  • 坐垫
    座椅坐垫一般有两部分组成：坐垫泡沫和坐垫而套。 坐垫泡沫是用来支撑、减震和缓冲作用，很大程度上影响乘员的舒适性。为了进一步提 供座椅的舒适性，可以将座椅泡沫做成软硬两层，软泡沫可以提高入座时的感觉，而硬泡沫 利于长时间的驾驶， 不容易疲劳。
    前排坐垫的角度普通车型定义在 13~15°，豪华车型定义为 12.5~14.5 °，普通车型坐垫 长度通常为 345 355mm，豪华车型定义在 365~375mrru 而后排坐垫的长度通常要求坐垫长度为300~335mm。从乘员角度来说，希望座椅的坐垫角度大一点为宜，来防止假人下潜在前碰中，坐垫较容易限制乘员髋部前滑，而实际是上，碰撞中，泡沫被压缩的很厉害,因此仅增加泡沫坐垫的角度，效果不明显，需要增加坐垫骨架， 和硬泡沬的倾斜角度。
    考虑到坐垫舒适性，对坐垫泡沬有压限量的要求。 通过座椅H点，并且平行于 XZ 平面的截面，坐垫最小厚度要求 85~91mm，压限量要求40~45%，坐垫外上表面到坐垫下支撑金属的距离（meattometal ) 要求不小于 50mm。（靠背泡沫压限量要求 35~40%，meat to metal 要求不小于 35mm )
    座椅坐垫面套分为三种：织物、真皮和人造革。 真皮面套一般用于高档豪华车型，或者高配车型，美观、大方、耐用，与人体接触透气性好，但价格昂贵，因此很多时候釆用真皮和人造革搭配使用。

座椅强度

碰撞中，座椅的强度，对乘员保护非常重要，这些座椅的变形都要尽可能地整改。上面是在具体试验中出现的座椅变形，为座椅骨架的静态变形，不允许座椅骨架出现上述变形形式，但在前期具体设计分析时，需要有个具体的量化指标来评价座椅的强度。在 50km/h 正碰和 64km/h 偏置碰工况下，对于 Hybridlll50th 的人体:

  • 座椅坐垫骨架的最大动态变形<40mm
  • 最大静态变形<35mm(相对于车身
  • 座椅滑轨及与车身连接支架不允许出现塑性变形
  • 座椅靠背骨架向前的旋转角<100°
  • 滑轨移动，对于电动座椅小于 32mm,对于手动座椅<15mm
  • 滑轨锁止机构不能出现失效；<15mm
  • 座椅变形中不能出现尖锐的菱角<15mm
可以用50km/h 正碰和 64km/h 偏置碰工况对应的滑台工况来进行模拟分析，一般来说同等情况下滑台工况要比整车工况恶劣点。当然座椅的强度不是越强越好。乘员的动能基本不是靠座椅变形来吸收的，但在乘员与约束系统作用力最大时，座椅的适当变形可以降低乘员伤害的峰值。

！！！下潜

乘员下潜(Submarining)是前碰中出现一种非常不好的现象，它是系安全带的乘员在前碰中造成严重的腹部损伤AIS30是一种典型的下潜过 程：乘员处于非正常或标准的坐姿，在前碰中，安全带的腰带从乘员的髋骨滑向腹部， 人的腹部无骨骼保护，有很多脏器，非常脆弱，腰带勒住腹部很容易导致腹部伤害，另外，乘员 的躯干下沉， 肩带勒住乘员的胸部，很容易导致胸部压缩量增大。所以下潜的危害很大

什么是“下潜”？
下潜可以描述为前碰时乘员髂骨上的安全带的腰带向上滑动，滑到髂骨的上方，滑进乘员的腹部。髋部和腹部的刚度对比，腹部要弱的多。

乘员的下潜可以通过对座椅的设计来避免。现在普遍的一种办法就是在座椅坐垫骨架 上增加防下潜支架，一个半座盆的坐垫骨架结构，半座盆通常有一个"斜坡"，在这个斜坡上端下面一般会布置一个防下潜梁，这根防下潜横梁加强了座盆斜坡上端的强度，当乘员的髋部滑到斜坡上端时，座盆能给髋部提供足够的支撑，不使髋部下沉，这样腰带就不会滑向腹部。真正起到防下潜的是座盆的斜坡面，一般座盆钣金厚度较薄，一般在O.8~1.0mm,强度不能满足髋部约束作用，所以在斜坡下增加防下潜横梁增强座盆斜坡上端的强度，鉴于此也有很多座椅在座盆斜坡下面增加一层钣金件来增加座盆的强度，例如奥迪A6是在座盆斜坡下面焊接了2.0mm厚度的钣金来增加座盆斜坡处的强度，防下潜的道理也是一样的在前碰中假人的髋部有向下的运动趋势，所以假人髋部的加速度往往是Z向高于 X 向。 通过大量的数据分析，假人前移髋部的运动轨迹基本是水平向前向下 20度的方向
所以防下潜相关的结构设计要能阻止假人髋部向前向下20度方向的运动。
根据经验，座椅座盆的斜坡的角度最好在30~45°。座盆要有足够的强度，尤其是在座盆斜坡上端位置，可以在此处增加横梁，或焊接钣金以增加其强度，座盆结构要满足上一节的强度要求，试验中防下潜结构不能出现图 1 3 1中的变形模式。以上讲的防下潜结构一般用于前排座椅，对于后排座椅也有防下潜结构，如奥迪Q5后排座椅防下潜结构是与坐垫集成在一起，而宝马 X4是集成在车身结构上。目前后排防下潜结构基本是这两种集成形式。现在有很多车型使用EPP泡沫来代替钣金结构作为防下潜支架，如图所示，这种结构一般与坐垫集成在一起，不但可以减重，同时也节约成本，同样可以起到防下潜的作用。无论哪种防下潜结构形式，与前排一样，在结构设计时要满足一定的要求。

后排防下潜结构的设计要求:

  • 防下潜支架斜坡角度30~60°
  • 坡度上端距离 R 点水 平距离155~185mm，垂直距离50~70mm
  • 在横向方面，防下潜支架宽至少 340mm。
  • 假人按照NCAP法规摆放，防下潜支架在横向上至少要能覆盖假人的髋部。

防下潜支架的强度并非越强越好，支架强度需要通过 CAE进行优化。例如，防下潜支架料厚为 0.7mm时，假人的胸部压缩量为 26mm，但将料厚增大为 1.0mm时，假人的胸部压缩量则为27.5mm,这说明防下潜支架过强对乘员保护性能也是不利的，所以要通过CAE或试验手段来匹配防下潜支架的强度。

R点和H点

R点是德国叫法，R代表 Reference, 即参考点，美国标准中叫SgRP ( Seating Reference Point 乘坐参考点）。在标准 SAE J4002 有 R 点 H 点设计确定程序，该程序被业界广泛采 纳，成为行业标准，它们有如下特点:

  • R点是车辆设计过程中早期就定义确定的参考点
  • 座椅移动可以产生很多 H 点，但只有唯一的一点叫R 点
  • R 点对整车人机设计非常重要，很多关键尺寸以它为基础来定义
  • R 点确定一般采用 95th 分位的人体模型

总的来说，对于某一个座位，其R点是唯一的，座椅每调节到一个位置就会有对应的H点，座椅调节的行程范围轨迹叫 Travebox。一般乘用车R 点、H点和TravelBox是右对称的，但也有很多是不对称的，例如，有些车辆驾驶员侧座椅是可前后上下调节，而副驾驶侧只能前后，不能上下调节

HPM 设备

HPD ( H-Point Design Tool ) 这是用于前期 H点设计用的工具，它根据 SAEj4004 确定， 与物理工具 HPM对应，由 SAE 以 IGES 格式发布。HPM ( H-Point Machine ) 是用来进行座椅 H点和座椅靠背角测量的设备，将座椅放在 设计位置，用 HPM测量出的 H 点坐标理论上与 R 点一致。HPM有两个版本，目前正在使用的是 HPM II, 在 SAEj4002 中有详细描述。HPM I 与之前的主要区别是：采用了单腿测量，在测量 H 点时，可以不安装小腿： 背盆采用三段式， 即胸盆、腹盆和髋盆，这样测量更精确。

安全带

安全带是汽车约束系统中最主要的保护装置，它是用来保护车内乘员，在紧急制动或交通事故中，它能将乘员牢牢地束缚在座椅上，合理地限制乘员的移动，很大程度上减轻乘员的二次碰撞，防止乘员被甩出车外。在现代的车辆上，安全带己经是标准配置，并且在交通法规中明确规定，乘员必须佩带安全带，这充分说明安全带的重要性。 在所有约束系统中，安全带对乘员的保护效果最好。当车辆在高速行驶时发生碰撞或紧急制动时，它能有效地约束乘员在车内移动，同时也能防止乘员在碰撞屮被甩出车外，也将车厢内发生二次碰撞减至最小。现代安全带理想的工作过程是：当发生碰撞或紧急制动时，首先，安全带会收紧， 在事故发生的第一时刻把人束缚在座椅上。然后，随着车辆减速度加大，人体受到安全带的束缚力也逐渐增大，这时安全带又适度放松，这样可以避免冲击力峰值，从而避免因拉力过大而使人肋骨受伤。现在好一点的安全带系统都带有预紧装置和限力装置。预紧装置在车辆发生碰撞时，会立即启动，将安全带迅速收紧，将乘客拉紧到座椅上，消除安全带与乘客之间的间隙（即安全的松弛量)，这样在接下来乘客运动中，乘客是在束缚力下前行，乘客的动能从一开始就得到消耗，这样在乘客整个前移中，避免了乘客无动能消耗的自由运动，使乘客的动能合理地被消耗掉。安全带限力装置的作用是，当安全带的张力过大，限力装置就会起作用，释放安全带，缓解乘客胸部负荷，减轻对乘员胸部造成的损伤。安全带还可以使乘员在交通事故中不被甩出车外。

安全带分类

对于常见的安全带系统，如果按照其固定安装方式来分，大致可分为：两点式安全带、三点式安全带和全背式安全带三类。

两点式安全带

根据约束位置不同，两点式安全带又分为安全腰带和安全肩带，安全腰带是最早发明的安全带形式，其两个固定连接点位于座椅下方，通过织带系在乘员的腰部，约束住乘员的重心使之不能前移，它的出现对保证乘员安全，降低乘员的伤害程度确实起到了很大作用。但一些事故结果表明，采用腰带约束，在撞车时，乘员头部及上躯体会绕腰、骨盆关节向前摆动，容易造成头部伤害，甚至有人证明了在某种场合下，佩戴腰带式安全带比不佩戴时伤害更加严重，因此这种安全带在前排座椅上己经基本不用，主要用在后排座椅上，随着汽车安全水平的发展，两点式安全带现在一般用于低端车型的后排中间位置。安全肩带又称斜挂式安全带 ( diagonalbelt),织带经乘员胸前斜挂在肩部，可防止乘员上体的过渡前倾。但由于在撞车时受力不均，下体容易先行挤出，导致小腿受伤，另外若安装不当，身体还会从带中脱出或头部被撞。这种安全带欧洲应用较多，国际标准虽然通过了这种安全带，但不推荐使用。通过开发膝部保护装置来与这种安全带联合使用，可消除上述缺点。

三点式安全带

顾名思义它有三个固定连接点，包括腰肩连续带和肩膝带两种。腰肩连续带是一种最常用的安全带型式，这种安全带综合了腰带和肩带的优点，既能限制乘员躯体向前移动，又能限制其上躯体过度前倾，作用效果较好，是前座乘员的主要约束形式，对于前碰、翻车有良好的保护作用，对后碰不起作用。最常用的典型的三点式安全带如图。

其他类型安全带

除此之外，还有全背式安全带又称马甲式安全带、五点式安全带。 全背式安全带是在两点式腰带上再连接两条肩带组合而成， 全背式安全带的想法来自赛车安全带，多了一条肩带，优点是可将碰撞时的载荷在胸部更大范围分散开来，对乘客肋骨、心脏和肺部的压力减少了大约一半左右。此外，现有三点式安全带最大的缺点是汽车翻滚或受到侧面碰撞时，乘员仍有可能从安全带中脱幵，全背式安全带则更能将乘员固定在适当的位置。其保护效果最好，也是最完善的一种，但使用不便，一般仅用于特殊用途车或赛车上，儿童座椅安全带也多用此结构。五点式安全带，一般用于赛车上，这种安全带保护效果最好，在发生碰撞时，能将乘员牢牢地束缚在座椅上，以免乘员被甩出去，或与其他部件发生碰撞。但使用不方便，对乘员过于束缚，不舒服。

锁扣

锁扣（buckle ) 是安全带总成的快速锁紧及松脱的连接部件， 带扣可设有调节装置，但全背式安全带除外。为了使用的方便，以及从安全考虑，法规对带扣有很多基本要求和规定。根据使用和布置的需求，锁扣也有各种各样的结构，有弹簧压片式的，还有棘轮锁止式的等。另外根据安装锁扣方式不同，带扣也有很多种类，织带型锁扣钢丝型锁扣刚性支架型锁扣。现在很多时候，在锁扣上集成了安全带预紧功能，这种预紧方式可以同时对肩带和腰带预紧，效率较高，而且成本也较低。所以，这种预紧式锁扣的使用越来越广泛。如宝马X4就采用锁扣预紧，省去卷收器预紧，能同时对肩带和腰带进行预紧。但这种锁扣预紧装置一般布置较困难。

另外，锁扣是柔性的织带或钢丝，还是刚性的钢板，对乘员保护性能有很大的影响。具体什么影响？

通过试验，在其他参数都完全相同的情况下，仅锁扣类型不同，分别是钢丝型锁扣和钢板型锁扣，结果反映在乘员胸部伤害方面，有很大的不同，钢板型锁扣乘员胸部压缩量要比钢丝型锁扣低近5mm。说明钢板型锁扣在有些约朿系统中对乘员保护还是非常有利的。从整个约束系统角度来分析这种现象产生的原因。安全带系统本身带有预紧功能，当采用钢丝型锁扣时，在预紧阶段，系统释放出很大的能量乘员胸部压缩量增加很快，因为预紧释放的能量无处吸收，累积并施加到乘员的胸部，在之后乘员胸部和安全带及安全气囊作用过程中，预紧阶段产生的胸部压缩量始终累积在乘员胸部，最终导致胸部压缩量峰值较大。而釆用钢板型锁扣，就不同，由于钢板型锁扣本身在横向移动需要较大的力，及钢板型锁扣在预紧时，要横向移动并且钢板要进行一定的变形，自然它就吸收预紧释放的能量，所以预紧阶段乘员的胸部压缩量下降并不大，从图中红色的虚线能明显看出，因此预紧阶段释放的能量在乘员胸部累积相对较少。

在正面碰撞中乘员有下潜的现象，在 2020 年EuroNCAP 新的评价场景 MPDB 评估中要对驾驶员侧假人 TH0R进行下潜评估，有 4分的修正。可以通过前期锁扣的设计来预防下潜的发生，对于前排安全带锁扣一般安装在座椅上，将假人按照MPDB评价试验协议调整到试验位置，并根据协议将安全带调整到试验位置，这时锁扣与X 方向的夹角大于 45°为宜，如图 2-2-9 所示， 锁扣布置的角度越大时，在假人向前运动过程中，腰带给髋部向下的分力也越大，这样可以限制髋部沿着座盆前移。

卷收器

卷收器是指安全带组成中用于收卷、贮存部分或全部织带，并在增加某些机构后起特定作用的装置，它是安全带系统中最复杂的机械件。卷收器的作用是在平时不用时将织带收藏起来，在使用时拉出即可，无需调节织带长度，乘员活动自如。发生碰撞事故时，又能锁住织带，以防止乘员因过分前冲而受伤。卷收器可分为无锁止式（NLR型）、自动锁紧式（ALR型）和紧急锁紧式（ELR 型）三大类。

无锁止式卷收器的功能只是收卷织带，安全带的长度调节机构要由另外的调节机构来 完成，其作用就是将织带全部拉出和卷入。自动锁紧式卷收器里面有棘轮机构，当织带从卷收器连续拉出过程一旦停止，棘轮机构就会自动锁紧，阻止拉出织带，使安全带不会自动放松。如果在拉出过程即使尚未达到所需长度而中途停顿，也会被自动锁紧，只能使其缩回重拉。因此，使用不大方便且带子的束缚力较大，舒适感差，通常用于腰带。

紧急锁紧式卷收器是当车辆遇到紧急状态时可将织带自动锁紧，而在正常情况下乘员可以在座椅上自由运动的一种卷收器。由于这种卷收器内的敏感元件按敏感对象不同，又分为车感型、带感型和兼之前面二者的双感型等三种类型。车感型卷收器的锁紧装置是由车身加速度传感控制的；带感型卷收器的锁紧装置是由织带拉出加速度传感控制的；双感型卷收器的锁紧装置是由车身加速度和织带拉出加速度的双重传感控制的。当检测到车体加速度超过极限值时，紧急锁紧式卷收器即发生锁紧作用。紧急锁紧式卷收器的主要性能指标有卷收力、紧急锁定敏感度（包括织带感应和车体感应）和耐久性要求。其中卷收力的主要测试有使用后的恢复性能（安全性）和使用时的压迫感（舒适性)。影响卷收力的因素有：弹簧的设计、卷轴的大小、卷收量、机构的性能和织带的性能。对于紧急锁定敏感度各个厂商都有不同的要求，对于合资车型，卷收力（肩带端测量）达到8N左右，且卷收力衰减慢，而对于自主的一些车型，卷收力要达到12N以上，在佩戴安全带时有不舒服感。

正常使用时，卷收器的功能就是将不用的织带卷进卷收器，要用时，随时将其从卷收器拉出，正常前排乘员使用的安全带系统织带总长度在 3500mm 左右，50th 分位人体正常佩带时卷收器内织带长度（也叫"卷残长度 WOSw ) 不小于 500mm。对于前排带有预紧限力功能的卷收器，在 NCAP 工况下，织带能从卷收器拉出 300mm 多。一般安全带有动态拉出量，因为安全带卷收器锁止机构必须在安全带织带拉出一定距离后方能锁止（锁止空行程），而卷收器锁止后因消除织带卷绕间隙（绕轴效应 film spool )又会拉出一小段，这些拉出量成为安全带动态拉出量，动态拉出量与卷收器的结构和制造而异，一般在 10~90mm 之间，在安全带动态拉出时，安全带几乎不能对乘员起到约束作用。当安全带系统的拉出量增大时，人体胸部的 X 方向位移增大了，安全带约束效率降低，这就是为什么目前大多数安全带系统中都装有预紧装置的主要原因，其目的就是要消除安全带与乘员间的间隙，使人体在乘员开始响应时刻即获得安全带的约束作用。当安全带对乘员的约束力过大，又应该适当减小这种约束力，减小对乘员胸部的伤害。因此，在许多安全带设计中还应考虑限力器的特性（下面将会详细讲解安全带的预紧和限力特性)。安全带的绕轴效应会影响安全带拉出的力一拉出量的关系。现在车型的安全带系统一般在卷收器上集成预紧和限力功能，现在所常见的：紧急锁止式，带有预紧和限力功能的安全带。

高调

安全带高调器 (High Adjuster ) 一般用来调节安全带上固定点的高度，使安全带系统能匹配不同分位的人体，一般带有高调器的安全带系统至少有三档调节，最高的用于高大的人体，中间用于中等身材的人体，最下端一般用于小个子的女性人体。在做 NCAP 前碰试验时，法规要求高调器放置在“设计位置〃，由于 NCAP 碰撞假人是 50th 分位的人体，所以很多时候，将高调器放置在上下中间位置，其实这个“设计位置"车辆制造商可以自己根据需要来指定，可以是最上面，也可以是最下面。

现在 NCAP 前碰工况中，一般是采用 H350th假人，这种假人，前碰中乘员的胸部最容易受到伤害，一般是胸部压缩量指标，假人的胸部压缩量测量是通过一个假人胸板上的一个探针来进行的，探针绕着下面的固定点转动，通过测量转动的角度来计算胸部压缩量值。

在实际试验中，该探针测量对安全带的佩带在胸部的位置很敏感，即安全带佩带远离探针，则胸部压缩量就相对较低，如果安全带佩带位置在探针位置上，则胸部压缩量就较高。 高调器处于最高位置时，安全带偏离了假人胸部探针，则同等情况下，测量出假人的伤害值要相对较低，反之亦然。鉴于假人这种规律特性，在进行碰撞安全幵发时，往往被利用来提高乘员的得分，即在做NCAP 评估时将高调器"设计位置"定义为最高位置，试验吋，将高调器置于最高位置， 有时候，这种方法能收到非常好的效果，例如某车型，将高调器置于最高位置比中间位置，朐部压缩量降低了3.3mm。这种方法需要经过试验匹配论证才行，一般来说驾驶员侧效果比较明显，而乘员侧往往不明显。

宝马所有车型，高调器都是不可调的，只有一个位置。

预紧和限力

预紧(Pretensioner) 和限力（LoadLimiter)功能是安全带系统最常用的，也是最重要的 两种功能，尤其对于前排安全带系统基本成为标配。在很多时候， 对于单预紧的安全带系统，一般将预紧和限力集成在卷收器中。

安全带预紧

预紧是安全带系统重要的功能，一般情况下，我们讲的“预紧”功能是指的被动安全预紧功能，即在车辆发生碰撞后安全带预紧功能。由于车辆碰撞发生的时间极短，例如64km/h偏置碰撞，核心碰撞过程仅150ms左右。因此安全带预紧功能一般是通过火药来驱动的。安全带预紧功能触发后，火药爆破推动钢球，带动与钢球啮合的齿轮旋转，将织带卷入卷收器内，缩短卷收器外的织带长度，这样安全带与乘用之间的间隙（松弛量）就基本被消除。

那么，安全带系统为什么要使用预紧功能呢？其实预紧功能主要是消除乘员与安全带之间的间隙（松弛量），尤其是冬天，穿羽绒服这种情况下，乘员与安全带之间的间隙过大。更需要安全带具有预紧功能。安全带系统是用来约束乘员的运动，在某特定工况的碰撞中，乘员的初始动能是一定的，能量公式是：E=F*d，即力乘以位移。在位移一定，动能一定的情况下，如果某段位移力为零或很小，那么在剩下位移上吸收剩下的动能就必须较大的力才行，这个较大的力往往导致乘员较大的伤害，因此从保护乘员角度出发，希望在整个位移过程中，力是均匀的，也就是乘员的动能最好能均匀被耗散掉，不希望在此过程中有"自由行程"。安全带整车佩带时，卷收器的卷收力一般只有几牛顿，安全带与乘员之间又有衣服相隔，因此，安全带与乘员之间往往有一定的间隙。在碰撞发生时，首先要消除乘员与安全带之间的松弛量，为下面约束系统保护提供前提。

有预紧时，一开始乘员胸部压缩量增加很快，耗散掉一部分动能，避免后期动能的集中耗散，所以后期峰值相对较低，而没有预紧的，前期能量吸收少，留到后期吸收，导致峰值较大。

安全带预紧有多种布置和集成方法，除了上面提到的与卷收器集成在一起，预紧功能还可以集成在锁扣上，这是宝马常釆用的布置方式，这种方式可以同时对肩带和腰带预紧。还有将预紧布置在安全带锚点上，这种预紧也叫端片预紧。

安全带限力

安全带限力是安全带另一个很重要的保护功能。限力是通过一根限力杆来完成的，即通过限力杆扭转变形使织带拉出力不会太大，避免安全带肩带力过大。

当限力杆转速为 3000 转/分时，限力峰值基本是恒定的，但当转速低时，比如 20 转/分，限力峰值就会稳步上升。

安全带限力在4KN，一般对应限力杆直径为 9.4mm。从乘员保护角度，安全带限力较低有利于试验中假人的得分，但由于 GB14166 中有假人前移量和速度的限制，因此该限力值不能太低，否则难以过法规，前排乘员限力（肩带力）一般 4kN 以上，后排至少在需要进行约束系统匹配而定。

织带在卷收器里面是缠绕在芯轴上的，因此织带有缠绕半径，如果适当增加安全带的卷残长度（WOS)则该半径就会变大，在限力杆扭矩一定的情况下，织带拉出力就会有所降低 因此如果想稍微降低一些限力值， 可以适当增加卷残长度来实现。

双级限力

对于不同的人体来说承受安全带的力是不同的，比如 95%的人，他们胸部能承受较大的力， 并且也能承受较大胸部压缩量，而 5%的女性承受的载荷就相对较小。因此，针对这种情况，人们将限力装置进行了衍化，出现了双级限力，就是将不同限力等级的限力杆集成在一起，以适应不同群体的人，如图 2 3 11 所示，这两个不同的限力杆代表了不同的限力等级，它们之间可以 switch 装置来进行切换，这样可以起到更好地保护乘员的目的。

双级限力根据需要分成两种：digressive 和 aggressive。digressive一般用于前排，第一级限力有利于增加对乘员的约束，第二级降低胸部的载荷，使乘员胸部载荷不至过大，而 aggressive —般用于后排，第一级用来保护儿童，第二级用来保护女性乘员。

安全带的双级限力需要通过试验来进行匹配， 尤其双级限力之间的切换时机。一般来说，预紧功能与限力功能一起使用，限力功能在价格上比预紧功能要便宜很多。

RGS（主动预紧安全带或者可逆式）

主动预紧就是在普通预紧产生之前先进行预紧，即还没有发生碰撞之前进行一定量的预紧工作。它的工作过程是这样的，在车辆即将发生碰撞时，车辆自带的传感器将检测到这一情况,然后触发主动预紧卷收器的电机，通过电机带动涡轮蜗杆使卷收器将安全带回收，从而实现安全带先行预紧，这一过程是发生在车辆碰撞之前，这个主动预紧是属于主动安全范畴，在可能发生车辆碰撞的情况下做的预防措施，如果车辆避开碰撞，那么这级预紧可以复位。当交通事故发生后,碰撞传感器采集到信号，触发第二级预紧， 这级预紧就是传统的预紧（火药驱动预紧），是被动安全的范畴，是不可复位的。

安全带布置

安全带是用来约束乘员的，所以安全带的布置也是为了更好地约束乘员，避免在碰撞由于布置问题带来乘员的额外伤害。关于安全带的布置，主要是校核其几个固定点位置的合理性,通过合理确定固定点位置，一方面匹配人机使用的方便性，另一方面使安全带对乘员的约束保护更合理。安全带的布置要能适应不同分位的人群。例如，有些安全带能缠绕（routing)50%，95%分位的人群，但对于5%的人群，安全带有可能出现滑肩现象。安全带肩带对女性乘员肩部缠绕不够，碰撞中有滑肩的风险。可以用相关专业的工具来评估安全带滑肩风险。抛开以上，还需要安全带固定点校核。
对于后排安全带的下固定点的布置要避免乘员可能下潜的风险。乘员下潜在前碰中是很常见的现象，尤其对女性乘员来说，容易出现下潜现象，后排安全带锁扣端离座椅表面太高，一般要求低于50mm，否则带扣容易上滑，导致乘员下潜

安全带聚集 ( bunching) 是安全带使用过程中出现的很不好的现象。于安全带在 D-Ring 聚集导致织带不能拉出，乘员的胸部载荷急剧增大，也有可能安全带被割断。无论哪一种，都将带来严重的后果。安全带聚集产生的过程:织带在 D-Ring 的槽内滑动，织带摺皱，直到安全带聚集。织带聚集是由于织带的横向刚度不够。 除了织带本身的问题外，安全带聚集与很多因素有关，例如，安全带系统的布置，D-Ring 的结构等。

在前期的安全设计阶段要避免安全带的聚集风险，解决安全带聚集问题，可以从以下几个方面着手:提高安全带的横向刚度，减少杜绝织带褶皱。对安全带系统进行合理的布置，这三个角度对安全带聚集有很大的影响。为了降低聚集的风险，需要：尽量使 a 角最小，使 0 角最大，使丫角最小。

改进 D Ring 的结构设计，提高 L 与C 的比值，一般来说，L较大可降低聚集的风险。

气囊

安全气囊也叫“辅助约束系统”( SRS Supplemental Restraint System ), 它是在碰撞中在乘员与车辆内饰间展开一个充满气体的气垫，通过“柔软”的气垫对乘员起缓冲作用。相对于安全带，安全气囊只是一个辅助保护设备，它是安全带的补充，在车辆发生碰撞时，对乘员起到附加的保护作用。安全气囊一般由气体发生器、控制系统和气囊3部分组成。安全气囊的控制系统主要由传感器、电子控制系统和触发装置三部分组成。传感器用来监测车辆的实时状态，将车辆的实时状态传输给电子控制系统，由电子控制系统对实时对当前状态进行判断，然后决定是否启动安全气囊。安全气囊最大的优点就是缓冲乘员头部的运动，防止与车内饰件发生碰撞，同时分散对乘员胸部的冲击力。

全气囊仅仅是“辅助约束系统”，它是一把“双刃剑”，使用恰当，它能保护人，如果使用不当，它会伤害人。安全气囊一般是通过火药来充气展开的，火药本身威力巨大，在向后展开的瞬间能产生约 800 公斤的力，气囊对乘员的伤害一般有两种形式，如图。如果没有与安全带配合使用，人过早接触气囊，会被气囊“打”伤，造成额外伤害。因此安全气囊一般在碰撞比较剧烈的事故中应用。在欧美国家，在气囊可展开，可不展开的情况下，一定不让气囊展开。

现代车型中气囊是基本配置，常见的气囊是驾驶员气囊 ( DAB), 乘员气囊（PAB )，侧气囊（SAB )，侧气帘（CAB )，膝部气囊（KAB )等， 我们平时所说的"六气囊配置"是指的:DAB, PAB, 2*SAB 和 2*CAB

基本保护过程

车辆的碰撞过程基本可以分为三个阶段：一次碰撞、二次碰撞和三次碰撞。一次碰撞一般指的是车辆和障碍物的碰撞，主要考察车身耐撞性能；二次碰撞是指乘员与乘员舱内饰件之间的碰撞， 主要考察约束系统对乘员保护的性能；三次碰撞是指的乘员器官之间的碰撞，这是生物力学的范畴。

对于现在研究的 NCAP 标准碰撞工况，整个碰撞过程基本在 150ms 就已经结束了，车辆的三个碰撞阶段在这段时间内全部完成。安全气囊就是在发生一次碰撞后，二次碰撞前，迅速在乘员与车辆内部结构之间打开一个充满气体的袋子，使乘员扑在气袋上，避免或减缓二次碰撞，从而达到保护乘员的目的。

当发生碰撞事故时，传感器感受车辆碰撞强度，并将感受到的信号传送到控制器，控制器接收传感器的信号并进行处理，当它判断有必要打开气囊时，立即发出点火信号以触发气体发生器，气体发生器接收到点火信号后，迅速点火并产生大量气体给气囊充气，使得乘员与一个较“柔软”的吸能缓冲物件相接触，而不是与车辆内饰件猛烈碰撞。如果将气囊在内部高压作用下冲破模块面盖的时刻定为 Oms, 约需 10ms 的时间气袋冲出，20ms 气袋已基本展幵，30~40ms 后气袋完全充满气体。然后气囊对乘员进行缓冲吸能，并且气袋逐步泄气，最后整个碰撞结束，整个系统卸载，乘员反弹，碰撞结束。注意，气囊展开的时间以及点火的时间根据不同的工况各不相同，本处只是举例，请不要用作真实工况的参考！

安全气囊最重要的指标是可靠性；如果不该点火而点火打开气囊称之为误点火；如果应该点火而没有点火称之为漏点火；如果点火太晚则称之为迟点火，无论是误点火、漏点火、还是迟点火都是不允许的。为了提高安全气囊系统的可靠性，防止电源线在碰撞中断线、电池遭到破坏，系统中备有储能电容或电池，以保证即使掉电也能够打开气囊。为了检测传感器、电子电路、气体发生器， 系统一般还有故障诊断模块，并设有信号灯予以显示。

驾驶员气囊

气袋

气袋是安全气囊的主要部件之一，由于其工作时间短，作用条件差，故对其质量要求非常高。其材料大多用锦纶或高强涤纶，要求细支、异型、多组份化，气囊织物要求轻薄、高强、气密、阻燃、高熔点、高热焓。且在面向乘员一侧有涂层，一般为淡红色或淡绿色，一方面降低织物的透气性，防止高温气体灼伤乘员，另一方面可避免气囊因长期叠置而成硬块，延长气囊的有效使用期，最长的可达到 15 年，基本同车辆使用寿命。

为了满足汽车安全气囊使用要求， 气袋织物需要具备很多性能，首先气袋要轻薄柔软，便于折叠；其次要能承受一定的充气压力，耐压性能的大小反应其安全保证的能力，一般气袋织物耐压性的标准为 1.7kg/cm2; 再有气袋织物不但要有较高的强度，同时还要有良好的伸长性能、较高的弹性和恢复能量的吸收性，以保证气袋膨胀后有弹性但不破裂；还要有良好的密封性能，气袋必须在短时间内充满气体，因此要求气袋织物透气量必须很小，另外也是其形状稳定性的保证，对于驾驶员侧，要求朝向驾驶员一面织物透气率小于 5L/(dm2min ),朝向仪表盘一面织物透气率小于 10L/(dm2min ), 而乘员侧要求气袋织物透气率为 10~15L/( dm2min), 要求整幅织物透气性均匀； 另外为了点火装置的热效应会导致气袋内，尤其是气袋进气口周围很高的热负荷， 要求气袋织物有一定的阻燃性能，FMVSS 302 法规规定，气袋织物燃烧率不超过 102mm/min,而 GB8410 规定，织物燃烧率不超过 100mm/min,或火焰能在 60s 内熄灭，且燃烧长度不超过 50mm。

对于企业内部，请参照VW气囊设计要求

DAB 气袋一般是平面气袋，即气袋可以在平面上铺平，由两片组成，上下缝合，它的容积一般在 40~60L，对于美式 DAB， 由于要考虑未系安全带乘员的保护和小偏置碰乘员的保护，因此美式气囊一般较大，有的会超过 60L。从乘员保护的角度看，相对较大的 DAB 气袋容积对乘员保护有利，但较大的气袋容积，会带来成本的增加，同时给发生器布置带来困难。对于一般的 B 级车辆，气袋容积在 50L 为宜。气袋厚度约为 0.3mm。

拉带

DAB 的拉带是用来控制气囊展开的包型，可以避免气袋向后对乘员的冲击，通过拉带使气袋与乘员的作用过程中，气袋位置稳定，不会滑向某一侧。DAB 气囊一般有 2~4 根拉带， 当气囊拉带为 2 根时， 有 3 点和 9 点的布置，或者 6 点和 12 点的布置方式。有时两个拉带不利于乘员的保护，在碰撞中，两根拉带的气囊容易受到驾驶员胸部的挤压作用，滑向方向盘上侧， 这样驾驶员胸部就失去了气囊的缓冲保护。

所以从乘员保护角度，希望气囊的拉带至少 3 根，四周成均匀分布，例如釆用了 3 根拉带，圆周均匀分布，其他参数不变的情况下，在对驾驶员约束过程中气袋减少上滑，驾驶员的胸部压缩能够降低2.5mm。

泄气孔

DAB 气囊一般有两个排气孔，排气孔的大小为0>15 35mm 之间， 排气孔的作用主要用来泄气，但排气孔有很多其他的作用:

  1. 减轻 DAB 的振荡。气囊展开后会上下振荡， 通过排气减轻振荡。
  2. 排气孔泄气，对外作功， 吸收乘员动能，降低气袋温度，避免气袋高温灼伤乘员。
  3. 排气孔泄气，降低气袋压力，使气袋变得更柔软。
  4. 气袋放气的过程，也是释放能量的过程。

在设计排气孔时，要考虑气囊展开周边部件的位置， 不能让其他部件遮挡排气孔。气囊的排气孔不宜过小，过小，气囊内部压力大，气袋很硬，对乘员保护不利；但排气孔的直径也不宜过大，过大， 气袋还没有真正缓冲约束乘员，气袋已经憋住了， 起不到保护乘员的作用。因此气囊的排气孔大小需要通过试验进行匹配，最终确定最佳的排气孔直径大小。

气体发生器

气体发生器 ( Inflator ) 与气袋一样都是气囊的关键部件，有效使用周期至少要达到 15年，前排气囊一般采用烟火式发生器。

气体发生器是危险的部件，内含火药，在漫长的使用中要保持高可靠性。要满足严格的认证标准，比较常用的标准是 USCAR24 标准，气体发生器使用的温度范围 40~85°C , 在该温度范围下，不能出现失效或降级。

高田气囊召回事件就是因为高田气囊发生器使用了廉价的硝酸铵，在漫长的使用中，硝酸铵容易受潮，受潮后片剂膨胀受损，出现相转移，片剂的表面积增大，导致爆炸速度威力增大，爆炸时，发生器会破碎飞溅伤人。

气体发生器必须做密闭容器试验（也叫 tank 试验)，不同的气体发生器要做的密闭容器试验容器尺寸所示。通过密闭容器试验可以测量发生器的颗粒物组成、质量，可以测量密闭容器的压力曲线，通过此试验可以得到 CAE 仿真用的发生器质量流，温度曲线。

在美标中，由于要考虑离位乘员（ OOP ) 的保护情况，因此美标的车辆， 气囊发生器一般是双级发生器（Dual Stage ), 传感器可以监测座位乘坐的是成人还是儿童，乘员是否系安全带，如果乘员未系安全带，或座位乘坐的是儿童，那么发生器第一级点爆，对气囊进行少量的充气。二级点爆一般用在在位乘员 (IP —In-Posltion )。

TTF（点火时间）

气囊的点火吋间（TTF—Time To Fire ) 是气囊对乘员保护的关键要素。如果气囊点火时间过早，气囊还没对乘员缓冲吸能，气囊就己经泄气，气袋没有足够的压力来支撑乘员，可能导致气袋"击穿 ( Bottoming-out )", 即乘员透过气袋与内饰部件发生硬性碰撞；如果气囊点火时间过迟，气囊展开时强大的冲击力，直接对着乘员，会带来额外的伤害。所以最佳气囊点火时间应当保证乘员在气袋刚好展幵到最大状态的那一瞬间，乘员与气袋接触。这样乘员不会被展开的气袋击打， 也不会因为与气袋接触过晚而没有得到缓冲， 甚至击穿气袋。

国际汽车界普遍采用的点火时间是：T5in-30ms 准则，用此准则来确定目标点火时刻，这个准则的含义是： 汽车碰撞过程中，乘员向前移动 5in (127mm ) 的时刻的前 30ms 时刻是气囊的目标点火时刻。

现在大多数乘用车驾驶员与方向盘之间距离为 420mm , 气囊充气后的厚度一般为 230mm, 气囊从点火到充满气体的时间为 30ms，这样当驾驶员前移 5in，即 127mm 时刻的前 30ms 为最佳点火时刻，如图。这里驾驶员前移 5in 是平均估值。上面提供各尺寸也是估值，最佳点火时刻的核心是气囊正好充满气体立即与乘员接触，这一时刻的前 30ms 为目标点火时刻。

上面讲了目标点火时刻确定的原理，但针对每一辆具体车辆在具体碰撞如何确定目标点火时刻呢，这是气囊 SDM ( 安全气囊控制模块）不停釆集车辆行驶信息，然后根据固化在 SDM 中的算法，再结合其他的条件最终确定气囊的点火时刻。气囊的点火控制算法是核心技术，一般掌握在安全零部件公司。基本上点火时间的算法有以下几种：

  1. 加速度法。这种算法是通过测量汽车上的加速度信号来实现的，当加速度信号超过预先设定的阀值就发出气囊点火信号，使气囊充气。
  2. 速度变量法。通过对测得的加速度信号积分即可得到速度变化量，当速度变化量超过预先设定的阀值时就发出点火信号。由于速度变化量比加速度信号平缓的多，而起能够反映碰撞过程的能量，所以这种方法有较强的抗干扰能力，目前正被广泛采用。
  3. 加速度坡度法。美国 ASL 试验室的 TonyGioutsos 提出了加速度坡度法。他认为加速度法抗干扰能力较差，而速度变量法由于对加速度进行积分而使得对碰撞强度不够敏感，所以提出釆用加速度变化量作为判断是否应该点火的指标。在求导之前先对测量得到的信号进行低通滤波，再对滤波过的信号进行求导得到加速度的坡度，当这个指标超过预设的阀值时就会发出点火信号。
  4. 移动窗式积分算法。它是对当前时刻得前w时间内得信号进行积分，得到一个指标S (t， w), 当 指 标 S ( t, w) 超过预先设置得阀值S 时，就发出点火信号，否则继续进行处理和比较。由于这种算法总是只处理最近的 w 时间长度内的信号，对窗外的信号自动遗忘， 因此称为移动窗式积分算法。

副司机气囊

乘员侧气囊 PAB 是用来在前碰中保护副驾驶员的气囊，发生器一般采用烟火式，对于美标气囊，发生器也多采用双级发生器。

副司机气囊一般布置在仪表板上，点火触发时，气袋冲破仪表板展开。但现在有些车型倾向于将 PAB 布置在上面风挡玻璃与顶棚之间，如图 3 3 2 所示，这样可以很好避免 PAB 展开冲击乘员，或打碎玻璃等不利情况。PAB一般是立体气囊，即不能在平面上铺开，由 2~3 片缝合而成。PAB 的体积一般是 70~120L, 但对于美式 PAB 体积较大，能达到 160L。为了控制气袋展开的包型，有时候也会在气囊内加入拉带。

布置

从 PAB 与仪表板配合结构来分,PAB 布置可分位有缝气囊和无缝气囊。有缝 PAB 布置的优点主要是成本低，工艺简单以及拆装方便，而缺点主要是气囊的盖板与仪表板之间的配合较难控制，包括间隙、面差和色差，导致外观很难令人满意。

无缝 PAB 布置最突出的优点是解决了有缝 PAB 布置的外观问题， 但它的成本大幅增加。为了满足气囊的起爆要求，仪表板表面材料不能太硬，需要采用专门用于气囊起爆的材料，并且整个仪表板本体，至少上本体，都需要使用该种材料；无缝气囊需要仪表板内表面做弱化撕裂线，需要投入一定设备，并为弱化工艺提供相应的夹具；为了按照气囊需要在仪表板上增加托架。虽然无缝 PAB 费用增加很多，但由于外观美观，因此现在市面上基本釆用无缝 PAB 布置方式。

PAB 的在仪表板上的布置位置来讲， 分前置式（Front-mount ) 和顶置式（Top-Mount )两种。这两种布置方式的主要针对气囊模块中心线沿 Y 轴旋转的程度来区分的，顶置式气囊中心线与Z 轴夹角小于 25 度，而前置式则大于 25 度，如图。一般来说，顶置式气囊是当下流行的一种布置方式，因为气囊起爆充气时不会对着乘员，而是斜向上，对风挡玻璃有一点冲击，然后利用风挡玻璃的反作用力，将 PAB 下压，处于乘员与仪表板之间，开始缓冲支撑乘员。

在进行 PAB 布置时，有些行业经验性的要求需要满足，如图 3 3 4 所示，气嚢中心线与风挡玻璃的夹角 A≤40 度，如果该角太大，气袋打击风挡玻璃在玻璃垂直方向的的分力就越大，玻璃破碎的风险就越大，对于 100L 左右的气袋该角度≤40 度；PAB 模块中心线与仪表板表面的交点到风挡玻璃的垂直距离 B≥150mm，如果该值不能满足那么需要旋转 PAB 模块，使得该值满足要求，这时顶置式气囊布置可能变成前置式气囊布置；PAB 底面距离仪表板横梁距离 C≥35mm; 气囊盖展开过程中距离风挡玻璃最小距离 D≥25mm。以上布置设计数值仅是工程一般经验值，具体还需要与 PAB 供应商讨论而定，因为不同的厂商的 PAB 展幵性能是不同的。

从 PAB 撕裂线方式来分，可分为：U 型盖板和H 型盖板，如图所示，H 型盖板需要特殊处理才能避免出现撕裂后尖角划伤气袋。

泄气孔

PAB 气囊一般有两个排气孔，排气孔的大小为3O~6Omm之间，但在实际使用中，排气孔大小、数量有很多种。有些要求高的 PAB, 会釆用主动式排气孔，这种排气孔会随着内部压力来调节排气孔大小。

侧气帘

侧气帘 CAB 现在是车辆的主要气囊配置，尤其是新版 CNCAP 中增加了侧碰头部保护区域的评估 (HPD), CAB 成了必不可少的配置。

现在 CAB般是前后贯穿布置，布置在顶棚和侧围上边梁之间。发生器采用烟火式或混合式，发生器有中置或后置布置方式。CAB —般没有排气孔，通过缝线间隙和织布孔来泄气。CAB 设计要考虑翻滚和不考虑翻滚两种设计，翻滚设计一般用于美标车辆，要求气袋保压时间长。CAB —般用于比较特殊恶劣的工况下，来保护乘员的头部。例如，翻滚、侧面柱碰。如果没有 CAB，头部伤害肯定大。新版 CNCAP 中有头部保护区域评估，需要配置 CAB。

无论是 C-NCAP 还是 EuroNCAP 评价章程，HPD 保护区域都是只评估 DLO ( Daylight Opening)区域，“ DLO 区域”出之 FMVSS201,是指将侧围可移动的装置移除、收起或摇下等，比如风挡玻璃等，然后用平行于 Y 轴的光线照射，穿过的区域即为 DL0 区域，因此 DL0 区域指那些在车辆侧面碰撞中可以暴露乘员头部的区域，像风挡玻璃区域、三角窗区域，由于在碰撞中玻璃碎裂， 导致乘员头部暴露，这些区域需要评估气帘等头部保护装置是否能覆盖。而像 A、B、C 和D 柱是固定的，头部在后面不暴露，本身起保护作用，因此这些区域是豁免的，不评估。

图中白色的圆角四边形为评价章程要求保护的头部区域，即头部保护装置（CAB) 必须能保护或覆盖此区域。从此图看，在 A 处侧气帘未能很好覆盖 HPD 保护区域（或覆盖余量不足），但 A 处处于C 柱 后，属于非 DLO 区域，是豁免区域，不评估。而该图中 DLO 区域内，侧气帘都能很好地覆盖 HPD 头部保护区域，因此该车型的侧气帘外观尺寸布置完全满足 C NCAP 和 EuroNCAP 头部保护区域的要求。

侧气帘内的缝线区尺寸也要满足要求（只评估 HPD 区域内的缝线区是否满足要求）:

  • 规则的圆形区域，要求半径小于 25mm
  • 条形区域，最大宽度小于 15mm
  • 不规则的圆形区域，取该圆形区域的长轴和短轴， 它们之和小于 100mm，如果该不规则的圆形区域与其他区域相通，则找到宽度为 15mm的位置，将其隔开，然后在在评估该不规则的圆形区域。

在实际评估时，一般将未展开的气帘点爆，然后用充气设备对气帘进行充气，C-NCAP是按照制造商推荐压力或 30~40kPa 压力充气，EnroNCAP 对充气压力无特别要求， 只要能将气帘充满气便于测量即可。然后用 3D 测量设备，记录侧气帘鼓包边界的位置，包括气帘内部的缝线区域边界。记录的边界点要足够多，以至于能在相关 CAD 软件中用这些点描绘出边界线。最后，在软件中对 HPD 区域、测量的侧气帘边界区域和 DLO 区域进行对比，最终做出判定。如图所示，为某车型前排HPD 区域、测量的侧气帘边界区域和 DLO 区域在软件中的对比，在软件中就能一目了然的对比侧气帘是否满足要求。

关于 HPD 保护区域评估修正，在 C-NCAP 中用于侧面可移动壁障碰撞 ( AE-MDB ), 而在EuroNCAP 中仅用于侧面柱碰评估，而不应用于侧面可移动壁障碰撞（AE-MDB) 评估中。

气囊控制器以及传感器

气体发生器

气囊控制器即Restraint System Diagnostic Module，简称为SDM。是一个带有外壳的实时电子控制单元，安装在车辆上，用来确保乘员保护系统的运作发生碰撞时通过相关传感器采集到碰撞信号，传递给SDM 通过固化的算法來确定是否点爆气囊，和预紧装置，确定何时点爆，最后发出点火指令。

SDM 在车辆上一般安装在中通道上，在换挡手柄之前的位置，安装方式隐藏，确保不受外界震动的影响，它不与车辆环境中任何部件有接触的机会，以免接受到错误的信号从而导致误爆，以下是安装布置的位置要求

  1. 首选的安装位置在车辆的中央通道上，通常在与驾驶员平行的位置,安装位置必须与车辆的纵轴相对称
  2. 对于有翻滚功能的 SDM，横向安装位置与中心线的偏差不能超过 10cm
  3. 特定的子结构上 SDM 的安装和固定，必须是金属与金属联结， 其他材料的组合必须要单独则测试。 在计划的车辆寿命内， SDM 外壳及子结构之间的压力必须得到保证
  4. 前碰时横向和垂向上的加速度以及侧碰时纵向和垂向上的加速度应该越小越好
  5. 碰撞发生时， 加速度信号从碰撞点到传感器位置的正确传送必须得到保证
  6. 误作用发生时， 车身中加速度信号的传输要尽可能的弱。 误作用中发生的加速度信号越小越好，并且要与上述的碰撞信号容易区分
  7. SDM 安装位置处的干扰信号(比如手刹传来的误作用信号） 要尽可能的小
  8. 实际安装模块感受方向与名义上的感受方向角度偏差不能大于 5 ° (在相应的测量方向上)
  9. 不允许将车辆其他组件放在或者同定在 SDM 及固定点上安装位置周围不能有外部物体， 粗糙的毛边以及焊接残留物
  10. 如果一定要用支架，那么不能有 2000Hz (不同的 SDM 要求不同） 以下的实质性的机械共振。 共振一定要用适当的方法衡量。 如果系统(模块与子结构） 存在 2000Hz(不同的 SDM 要求不同） 以下的共振， 就要独立评估共振对系统的影响。 会产生共振的车辆零件(比如电动机） 不能装在离 AB-BCU 安装位置近的地方!注意不同的车辆配置
  11. 除了安装点外,SDM 外壳与车身结构不能接触。用来表示位置的针脚（如果有的话)也不能与车身结构有接触
  12. SDM 的安装不能在或者靠近碰撞发生时的车身结构破裂点
  13. SDM 外壳四周和车辆其他部件之间必须有足够的空间(最小 6mm)
  14. 在 50-2000HZ 的频率范围内， 安装位置的局部动态坚硬度要大于 2x106 N/m。 在此 频率范围内， 这个值的变化要小于平均值的 50% , 局部静态坚硬度不能小于3x106N/m
  15. 外壳和子结构之间的联结不能导致子结构的变形。 在安装过程中或安装以后， 模块 不能变形或损坏。 避免安装在不同的结构上(比如:两个单一的支架)
  16. SDM 不能与液体及污染物接触， 相对于安装位置的情况屬块的保护级别必须足够
  17. SDM 不能直接放在空调的气流中
  18. SDM 不能放在中央通道的凹处
  19. SDM 被固定后， 允许插上和拔去相关接头

以上 SDM 布置要求， 只是一般的布置要求， 具体以 SDM 供应商要求为准。

气囊传感器

碰撞传感器是用来实时采集碰撞信号的。
对于前部碰撞传感器， 一般布置在前防撞横梁安装板后， 或纵梁前端， 该位置对碰撞 信号敏感， 而且不易被触碰到。 但有的车型前悬过长， 在防撞横梁安装板位置， 传感器采 集的碰撞信号不强， 碰撞开始脉冲没有激励起来， 使得安全气囊、 安全带点火时间过晚， 这种情况往往需要重新选定碰撞传感器安装位置， 这样线束、 支架等需要更改设变。 所以 在车辆前期设计阶段， 需要论证碰撞传感器的布置位置， 以免后期设计变更， 成本增加、开发周期延长。

对于侧碰传感器一般布置在 B 柱下端， 前后布置在轮罩附近。

气囊零部件试验

气囊是辅助约束系统， 它必须与其他约束系统配合使用， 比如安全带， 才能发挥其保护 作用， 因此气囊系统在装配整车前要进行各种严格的匹配和试验测试。

气囊零部件试验主要有三种： 压力容器试验 （ tank)、 动态冲击试验和静态点爆试验。

压力容器试验， 也叫 tank 试验是针对气体发生器必做试验。 通过 tank 试验可以测量发 生器的颗粒物组成、 质量， 也可以测量密闭容器的压力曲线， 通过此试验可以得到 CAE 仿真 用的发生器质量流m（t）, 温度曲线T(t)，。 通过 tank 试 验可以测量发生器质量流曲线， 能清晰地看出气囊的充气量。 下面为某个 CAB 气囊的 tank 试验质量流曲线。 一般来说很难直接测量发生器的质量流m(t)， 可以通过测量压力Ptank(t)、温度T(t)等， 然后由公式计算所得。 一般Ptank(t)、 m(t)、 T(t)由气囊供应商提供给主机厂。

气囊总成的动态冲击试验有两种： 跌落塔（ Drop Tower) 和线性冲击， 。 动态冲击要选择合适的冲击速度， 例如正面 50km/h 刚性墙碰撞， 人体对气囊的冲击速度大约 4~8m/s, 那么可以选择这个区间速度的人体模块对气囊进行冲击。 对于跌落塔试验是使用重力的作用使冲击模块达到目标速度， 有时冲击模块悬挂高度达到 2 米以上， 模块运动的轨道较长， 实际运动中又有摩擦作用， 这样影响最终冲击速度的准确性， 会造成气囊冲击接触过早或过晚， 无法得到完整的气囊动态冲击响应数据。而气囊线性冲击方法是将冲击模块用液压装置加速到要求速度， 然后释放， 与展幵的气囊模块冲击接触， 这种方法冲击目标点准确， 速度控制更加精确等。 无论采用哪种动态冲击方法，它们的试验目的基本是：

  • 验证不同体积、 排气孔的气袋内部压力的变化
  • 验证气袋材料的强度， 气袋的泄气性（ 保压性 )
  • 获得气囊的动态冲击响应数据， 用于气囊模块的 CAE 对标

对于不同的气囊， 不同的冲击块质量， 以及不同的冲击速度， 气囊的动态响应是不同的,因此要根据气囊保护的人体不同部位结构和质量来确定冲击块， 然后根据气囊冲击响应来评价气囊模块保护性能的优劣。 保护性能好的气囊模块， 加速度明显， 走势平稳， 有稳定的脉宽， 没有突然变化的尖峰； 如果响应加速度有突然增大的尖峰， 说明气囊偏硬； 如果沖击块前期加速度偏低， 后期突然增加， 则气囊泄气过快， 气囊偏软。 这种偏软或偏硬的气囊对乘员保护都不利。 如下图所示， 实线的气囊加速度响应平稳， 有稳定脉宽， 没有突然增大的尖峰， 有很好的保护效果， 而虚线的气囊加速度响应前期偏低， 后期突然增大， 说明气囊偏软， 泄气较快， 气囊被击穿， 这样气囊保护效果差。

气囊的静态展开试验的目的主要有两个:

  1. 考察气囊展开的状态的稳定性、 充气时间
  2. 气囊展开时， 是否有飞濺物， 飞溅物类型， 以及飞溅物的质量

气囊展开时， 有时气袋会夹在内饰件之间， 导致气袋不能正常伸展，例如侧气帘有时会卡在顶棚和护板之间，导致气袋落不下来；有的气囊展幵需要撕裂内饰件，例如 DAB 和 PAB 等，有时撕裂口会产生尖锐的棱边，会划破气袋; 气囊气袋展开冲出的阻力不能太大， 否则影响气囊充气时间。

关于气囊飞溅物的要求参考GB/T19949-2005 道路车辆安全气囊部件

气囊静态展开试验中， 与正常坐姿的乘员相接触的气袋表面部分不能破裂或燃烧， 其他部分允许有直径小于 3mm 的破裂小洞， 气袋撕裂缝处应该保持完好无损。

仪表板

从广义上讲，仪表板 IP 也属于约束系统的一部分，它在碰撞中可以缓解减轻二次碰撞的程度，吸收二次碰撞的能量 ，直接保护乘员。因此 IP 的设计对乘员保护非常重要，在有些 NCAP 法规中对 IP 有要求。

保护原理

在碰撞过程中， 车身结构与外界壁障碰撞为一次碰撞， 一次碰撞是为了使乘降 (Ridedown) 最大化，即车身结构最大限度吸收碰撞动能，而使碰撞动能转移到乘员身上最小，然后二次碰撞通过约束系统来进一步吸收乘员动能， 因此在特定的工况下稳定的车身结构， 乘员的碰撞总动能是一定的，根据物理学知识， 如果使乘员受到的伤害最低，那么乘员的动能最好能均匀地被耗散掉。所以乘员在碰撞过程中身体各部分受力要均衡， 使乘员保持一个良好的运动姿态， 这就需要通过约束系统各部分互相匹配实现。

再来考察乘员受力， 乘员某部分受力，太大不行！太大容易导出受力的身体部位伤害加大；太小也不行， 由于乘员身体总动能一定， 某部位受力太小，其他部位受力相对变大。例如，IP 给乘员膝部和小腿的力， 如果该力太大， 则容易导致乘员下肢骨折，但另一方面，乘员膝部受力大，大腿则受到向后的轴向力也大， 该力传递到乘员的躯干，这样乘员的胸部受到载荷就相对降低，胸部受到的伤害也相对降低。有些车型布置了膝部气囊 KAB，一方面可以缓冲膝部， 避免载荷较大峰值， 同时大腿的平均载荷增加， 缓解了胸部载荷， 提高了胸部得分

碰撞安全乘员保护方面， IP 的设计要考虑到两点:

  1. IP 的布置及材料要合理，不能乘员膝部和小腿碰到硬物，也不能使乘员下肢受力偏高
  2. IP 要对乘员下肢耍有吸能缓冲的功能， 前碰中 IP 要给乘员下肢有足够的支撑和缓冲

在 EuroNCAP 法规中针对大腿膝部的这两个修正， 在评估中很容易失分， 而且分值较 大， 因此在车辆安全设计的早期要充分考虑避免这两项修正。

前期设计时，首先要确定评估考察区域，鉴于法规的评估考察区域的定义，在设计 时，该区域要留有足够的余量，建议区域上下 Z土100mm，纵向 X-40mm ( 考虑 40mm 穿透)。 同时考虑防火墙 70mm 的侵入，CCB 20mm 的侵入，即将制动踏板带着假人脚平面后移 70mm, IP表面后移 20mm，然后将假人（ 50"'人体）髋部前移，直到膝部与 IP 接触。以此接触点来确定评估考察区域。

区域确定后， 首先在该区域内进行布置校核， 该区域不能有布置的硬点， 例如， 电器盒， 支架等。 有些布置是无法避免的， 比如转向系统调节锁止手柄， 最好将其设计成可断裂的塑料结构， 即受到撞击时， 能断裂脱落。

对于转向管柱下护罩，在前碰中，假人的膝部很容易接触到，为了防止修正，护罩内恻最好能布置吸能缓冲结构，一般的做法是，在护罩内侧布置一层厚约 20mm 的泡沫 （ 至少10mm 厚），泡沫层外侧贴合一个铝的金属板，如图所示，这样既能缓冲，也可以进行吸能。

还有一个避免修正的很好的方法，就是保证乘员膝部与 IP 之间有足够的空间，即膝部最大前移情况下膝部与IP仍有20mm的水平距离，这样可以完全消除修正。在设计前期要考虑，必要时可以考虑安全带双预紧，限制乘员髋部前移 在 IP 膝部区域贴上一层泡沬，X 向 20mm，试验中，如果泡沫上没有漆印，则证明乘员膝部与 IP 至少有 20mm 水平距离，这样膝部就不会应用修正。但这种做法，给布置带来很大的困难，也增加了成本， 同时， 也给乘员其他部位保护提高了难度， 比如胸部。 所以是否使用该方法，要综合考虑。

另外，还要考虑膝部区域IP的表面造型。在评价小腿伤害方面，有一个最重要的指标小腿弯矩 My, —般小腿失分多，多与此弯矩偏高有关。乘员的下肢可以看着是杆状结构(大腿、小腿、脚)，杆子之间通过运动副铰连接（膝关节、踝关节)，在膝关节和踝关节之间的小腿可以看着“ 二力杆” ，正常情况下，如果只在膝关节和踝关节对小腿施加载荷的话，小腿只受轴向力，但如果在两个关节之间对小腿施加非轴向载荷，那么小腿就会产生弯矩（ 理论是如此)。小腿在向前运动过程中，最好不要与 IP 或其他内饰接触，为了做到这一点，最好在静止状态下，膝部与 IP 距离要小于小腿与 IP 距离，这样膝部先与IP接触，避免或减轻小腿与 IP 接触产生的弯矩。所以在前期膝部区域 IP 表面造型时要考虑表面的倾角。如图所示， 一种是好的设计， 一种是不好的设计。

膝部区域的 IP 表面造型在横向上过度不能过于倾斜， 如果过于倾斜， 膝部接触到 IP表面就会出现滑动，不但会碰到硬物，也会导致小腿运动的不稳定性，增加小腿伤害。

地毯

地毯组成

  1. 脚踏垫,材料一般为 PVC 或 TPO, 要设置在驾驶员侧，因为此处的使用频率最高，可以起到有效的耐磨作用
  2. 面料层,可以分为针刺地毯 (PET), 簇绒地毯（ PA/PP) 与其他塑料地毯（ PVC), 最常用的就是针刺地毯。在豪华车上一般使用簇绒地毯，显得更加高档。PVC地毯材料的一些挥发性能不是十分优良的话，对轿车的整体性会产生一定影响，因此只有在少量低档车与专用车上使用
  3. 背涂层,支撑面料的作用，优化地毯性能的作用，为了使地毯更好地成型，一般会使用PE或EVA材料的涂层，这些材料对隔声、隔热也起到一定的作用
  4. 填充层,此层一般根据不同的设计要求， 会以 PU 泡沫， 废纺毡与毛毡材料进行填充,起到隔声、 吸声作用， 以达到改善 NVH 性能的作用， 同时也有舒适的弹性， 以提升整车档次

安全相关

前部碰撞中，乘员的下肢在惯性作用下，脚沿地毯或加速踏板向前滑动，如图所示，膝部前移与 IP 接触，由于乘员的下肢可以抽象成通过铰链连接起来的杆状结构，“ 铰链” 的膝关节和踝关节没有约束，因此随着下肢前移与前面 IP、防火墙、地毯坡度，或者转向机构接触时，下肢的运动姿态会发生很大的变化，例如，膝部与 IP 接触后沿表面向左或向右滑动，脚踝受力后，可能向左或向右“ 崴脚” ，这种运动姿态带有很大的不确定性，即使相同的车型，两次相同工况的试验，下肢的运动姿态都会有很大的差别。

由于前碰中，乘员的下肢运动姿态的不确定性，因此采用 CAE 仿真手段也不能很好模拟碰撞中下肢运动情况。所以在 NCAP 评估中，乘员下肢最容易遭受伤害，而且在目前的技术情况下，对下肢伤害分析、保护一直没有较好的办法。从乘员保护角度出发，在前碰中，乘员的下肢需要保持稳定的运动姿态，尤其不能出现“ 崴脚” 现象。

这里涉及到另外一个问题，就是在试验中如何摆放假人的脚，同样会对结果造成巨大影响。 对于地毯来说：
乘员的脚置于地毯上，为了在前碰中保持下肢的运动姿态稳定，可以通过优化地毯设计来限制脚的滑动，在地毯的脚踵点位置做一个“凹坑”， 如图所示，当脚正常摆放时，脚后跟正好处于这个 “凹坑” 内（ 这种造型一般是针对悬吊式加速踏板布置)，这样在前碰时，这个“ 凹坑” 可以限制脚后跟前移，使脚的运动保持稳定，也能很好地避免脚前“ 甩” 使胫骨碰到IP( 避免小腿胫骨弯矩 My 偏大)。

另外，可以在地板或地毯上做一个Heel Pusher装置或造型，原理也类似，都是为了限制脚部向前运动，避免弯矩 My 偏大，同时也可以通过这个Heel Pusher来缓冲脚部载荷，减小小腿的肢骨轴向力 Fz，所以 Heel Pusher 装置一般是在地毯下布置泡沫来实现的，泡沫的大小可以覆盖脚底板的前后，横向区域，它的强度和硬度可以通过 CAE 进行优化，确保在碰撞过程中不能碎裂，能变形吸能，一般依据工程经验要求泡沬密度不低于80g/L, 厚度至少30mm，具体需要结合工程实际进行优化。

加速踏板

在前碰中，加速踏板直接与驾驶员脚作用，尤其 NCAP 评估中，驾驶员的一只脚放在加速踏板上的，因此加速踏板对乘员的下肢伤害有直接的影响。所以本章讲解加速踏板对乘员保护的影响，及其设计。

踏板形式

  1. 悬吊式
  2. 地板式

对于悬吊式，优点：

  • 不挑姿势，在任何坐姿下都可以很轻松的踩油门
  • 脚感轻
  • 成本低、易于平台化
  • 从安全角度讲，方便小腿空间的布置
由于踏板是悬吊的，因此地毯的布置空间很大，比如可以增加吸能泡沫，布置气囊，布置凹坑等。

缺点：

  • 不方便做跟趾动作
  • 踏板面积小， 脚感不好
  • 不利于实现油门的精确控制， 踏板支点在上，而脚的支点在下， 两者运动别扭
  • 驶容易疲劳，踩踏板时， 使用到胫骨前肌，由于悬吊式踏板比较轻盈，无法承受脚板自然伸展的重量，为了控制油门，需要时刻绷紧前肌

对于地板式，优点：

  • 方便做跟趾动作
  • 脚感好， 整个脚掌可以放在踏板上，踏板运动与脚的运动一致
  • 油门的控制通过小腿比目肌来实现的，不容易疲劳，小腿下侧是比目鱼肌。从油门踏板到制动踏板是胫骨前肌发挥作用，而从制动踏板到油门踏板是比目鱼肌发挥作 用，进行油门控制时， 几乎用不到胫骨前肌，只需要比目鱼肌伸展踩踏，脚部支撑是通过踏板反作用力来支撑，所以地板式踏板一般显的很“沉重”，从生物学来讲，伸展脚容易，而翘起脚费力，这是因为比目鱼肌比胫骨前肌体积大
  • 更好地使用 “kick-down” 按钮

缺点是：

  • 不适合坐姿高的车型，因为采用地板式踏板车型坐姿偏低
  • 在前碰中，对下肢碰撞安全不利

以后车基本偏重于家用，家用车比较讲宄操控轻便，易转向，车头短，车内舒适，坐姿高，坐姿增高后，便于驾驶者踩踏板，以及考虑踏板前后移动距离，所以更倾向于悬吊踏板。从目前的发展趋势看，地板式加速踏板形式在减少，大众多款车型逐步切换到悬吊式踏板，例如，髙尔夫 6 以后都采用悬吊式加速踏板。另外，现在很多高档车都用悬吊式，例如，奥迪几乎全系采用悬吊式加速踏板， 宾利， 沃尔沃，保时捷，奔驰很多高级的 SUV也用悬吊式加速踏板。BMW 全系都是采用地板式加速踏板。

安全相关

从乘员保护的原理触发，乘员下肢在运动中最好能保持稳定。如果釆用悬吊式加速踏板，便于在地毯上进行造型，或 Heel Pusher 布置，采用了悬吊式加速踏板，可以在地毯上布置 Heel Block, 这样在前碰中限制乘员脚前移。而地扳式加速踏板占用空间大，布置在地毯上，因此要求脚部区域的地毯是平坦，在前碰中不利于对脚部的保护。

转向系统

转向系统是汽车的重要总成部件，它承载着很多功能和性能的需求，对于碰撞安全的乘员保护来说，转向系统也是非常重要的，在前碰中，转向系统通过安全气囊与驾驶员作用，因此转向系统性能对乘员保护影响也很大。

方向盘

方向盘由于轻量化的要求，其骨架一般采用镁铝合金。方向盘通过花键螺栓与转向管柱相连。在方向盘中间集成了驾驶员气囊（DAB )，发生碰撞时，DAB 从方向盘中间展开，而方向盘盘面支撑气袋，对乘员进行缓冲约束。

考虑离位乘员保护（Out of-Position)，即未系安全带，非正常坐姿的乘员，在前碰时，或急刹车时，驾驶员会趴在方向盘上，如果方向盘上的 DAB 模块不好的设计过于突出，则会对乘员胸部产生伤害，所以在设计时要考虑 DAB模块的布置，根据工程经验，模块距离方向盘表面距离至少10mm,如图所示。

转向管柱

转向柱组成

转向管柱系统通常采用3段设计，即转向管柱、中间轴和传动轴。中间轴分为两段，它们之间是通过花键连接，彼此可以轴向相对滑动，一般有一定的滑动阻力，大约 70N。当碰撞发生时，前端转向机受到撞击，载荷传递给传动轴，传动轴再通过万向节传递给中间轴，由于中间轴是两段，彼此是可以轴向运动的，因此前舱碰撞载荷传递在中间轴上就断开了，不能传递给方向盘，这样避免乘员受到转向系统后侵的额外伤害。

转向系统的中间轴的设计可以避免转向系统的侵入伤害，但碰撞中，驾驶员扑向方向盘转向管柱通过方向盘气囊对驾驶员胸部作用，导致乘员胸部伤害。

转向柱压溃原理

在前碰中，驾驶员胸部一般受到安全带力、管柱通过气囊的支撑力，这两个力共同作用驾驶员胸部，从胸部压缩量时间历程曲线可以看出，胸部压缩量逐渐迅速增大，安全带在限力作用下，在某时刻达到最大值，且相对稳定，这时胸部压缩量继续增加，很大部分是因为管柱对胸部的作用力所致。如图中的蓝色实线所示，在管柱无压渍功能情况下，胸部压缩量一直增大到峰值，然后碰撞基本结束，约束系统载荷卸载后，胸部压缩量才逐渐降低。如果在胸部压缩量未达到峰值之前,管柱突然〃后撤"，带动方向盘、气囊向前移动，减少对驾驶员胸部的支持力，那么驾驶员胸部载荷就会部分"卸载"，胸部压缩量会逐渐降低，如图中虚线所示，这样胸部压缩量峰值就会降低， 从而达到保护驾驶员的目的，这就是转向管柱压溃保护驾驶员的原理。

在管柱压溃后，胸部压缩量降低，但随着管柱压溃行程的结束，整个压溃运动触底，被阻挡停止，在惯性作用下，管柱对驾驶员胸部的支撑力又突然变大，因此胸部压缩量在后期会出现第二个峰值。这个峰值与管柱压溃行程及压溃力有关。一般来说，对于压溃行程长的 (例如 70mm 以上），第二个峰值相对较低，煉 而压溃行程较短的（例如 50mm 左右），那么第二个峰值相对较高，有时会高于第一个峰值，因此这种情况要避免的。在可能的情况下尽量将管柱压溃行程设计长一点。根据工程师经验，管柱压溃行程至少为 50mm 才行。

转向柱压溃行程

管柱在压溃过程中，通过一个滑槽来定义管柱的压溃行程，如图所示，管柱上面有一个滑槽，滑块从上到下滑动，滑到底，表示管柱压溃行程结束。管柱压溃行程有很多设计方法。在转向系统压溃行程设计好后，在布置时，还要进行管柱压溃行程校核工作，因为在压溃行程上还可能存在其他部件阻碍管柱的压溃，这样使设计的压溃行程，实际中达不到。因此在管柱压溃的行程上，不能与任何部件发生运动干涉，即使塑料件也不行，因为很多塑料件本身不但有足够的强度，而且韧性很大，影响压溃。另外在管柱压溃中，转向管柱上下护罩和组合幵关是随动的，因此它们也不能与其他部件有运动干涉。如图所示，管柱上护罩与管柱安装支架发生运动干涉，必须进行整改。

管柱的压溃力分两个方面，一是管柱幵始压溃时机构的剪断力，这个力一般较高，二是压溃进行中压溃的保持力。如图所示，压溃幵始时，首先要克服图中的剪断机构的剪切力，然后压溃过程中，压溃力通过一个螺旋的钢带来实现的，这种功能有好多种实现方式，如可以用钢片的撕裂力来实现。

转向柱压溃力

管柱的压溃力大小对乘员保护性能影响很大，试想一下，如果压溃力很大，可能在碰撞中导致管柱压溃不了，如果压溃力太小，驾驶员胸部则失去了管柱的支撑力，胸部只有安全带来约束，这对乘员保护不利。理想情况下，乘员胸部应该受到各系统的均衡支撑约束，这样才能最大限度降低乘员的伤害。所以管柱压溃力，需要通过 CAE 或试验手段进行确认。管柱压溃力可以通过试验获得， 压溃力分静态压溃力和动态压溃力。

管柱静态压溃时，将管柱调节到中间位置，高度调节到最大拉伸位置，在模拟方向盘的中心处施加轴向的载荷，加载速度 100mm/min，允许管柱在轴向弯曲，加载到压溃行程结束。管柱静态压溃力一般较低，约在2.5~3.5kN。如图所示，为管柱静态压溃试验。管柱静态压溃力对于乘员保护性能分析意义不大，在分析及 CAE 仿真时，还是需耍使用动态压溃力，

在管柱动态压溃试验中，管柱至少要能压溃设计行程的一半，管柱 X，Y，Z 方向的压溃力超过 7.5KN 不能超过3ms。

转向柱护套

在前碰中，转向管柱护罩经常与驾驶员膝部发生碰撞，在前期布置设计时，也要注重护罩的设计。护罩内侧最好能布置吸能缓冲结构，一般的做法是， 在护罩内侧布置一层厚约20mm 的泡沫（至少 10mm 厚）泡沫层外侧贴合一个错的金属板，这样既能缓冲， 也可以进行吸能。

转向系统调节手柄在碰撞中往往会碰到乘员的膝部，导致集中载荷的出现，参加 EuroNCAP 评估的车，调节手柄采用金属材料，手柄突出并处于膝部碰撞区域内，在偏置碰中，膝部碰到手柄，导致集中载荷，大腿压缩力达到 4.5kN!因此，调节手柄最好不要布置在膝部碰撞区域内，如果无法避免，那么要将调节手柄设计成可断裂的材料，比如塑料材料，在受力时可以断裂。

生物力学

生物力学（Biomechanics) 对于刚接触碰撞安全不久的工程师来说可能很陌生。但在整个碰撞安全研发体系中是极其重要的环节，可以说，它是碰撞安全研宄和开发的基础，没有它我们无法进行碰撞安全的开发工作。

乘员在车辆事故中受到的伤害是多种多样的，有的是擦伤流血，有的骨折，有的是头部受伤，有的是下肢受伤，有的是内脏受伤...对于这些不同部位，不同形式的受伤，我们不可能具体地研究某一个种伤害情况，因此要对事故中的伤害进行归类，将伤害统一起来，这样才可以比较不同部位，不同形式的伤害之间的严重程度，换言之，就是将伤害划分等级，将不同部分，不同形式的伤害进行数字化。进一步地， 研究造成这些等级伤害的原因是什么。例如，是冲击造成的，还是挤压力过大等，然后再研宂每一种造成人体伤害的具体参数，例如导致某部位的骨折，那么多大力才能导致骨折呢，也就是导致骨折的阀值。当将乘员的伤害原因和伤害阀值搞清楚后，这样我们才能制造出 ATD( AnthropomorphicTest Device ) 设备来模拟真实的乘员，通过传感器计算，测量相关数据，与之前乘员伤害的阀值进行比较，通过这个来分析判断，车辆将来在事故中造成乘员伤害的程度。

因此生物力学的研究其实是在〃碰撞安全设计〃与"交通事故中伤害"之间建立了一座桥梁，如图111所示，对来自实际交通事故中的乘员伤害进行归类统计，进行生物力学研究，找到损伤的机械响应， 损伤机理，测量乘员损伤的耐受度（Tolerance ), 然后基于这些，我们制造能模拟乘员的伤害的 ATD 假人模型，通过 ATD 来进行车辆的安全设计工作。

对于主机厂来讲，主要工作在"安全设计"这模块，其他模块均有专门的机构组织来完成。

生物力学是一个很广泛的概念，我们车辆碰撞安全研究的生物力学仅是其一个分支。我 们车辆碰撞安全研究的生物力学包括四个方面:

  • 人体的损伤机理
  • 人体的损伤机械响应
  • 人体的损伤耐受度
  • ATD 设备

这里仅针对前三部分进行一定程度的展开和讲解。

碰撞生物力学

碰撞生物力学是碰撞安全研究的重要基础理论之一，在碰撞中，人体暴露在一个机械冲击载荷的环境中，在惯性力和接触载荷的作用下，人体各个部位的组织产生一定的生物力学响应，如果这个生物力学的响应超过人体组织可以恢复的限度或导致解剖学组织破坏，导致正常生理功能的变化及丧失，这就产生了人体损伤，因此碰撞生物力学是研究人体组织在碰撞过程中涉及的力学性能。

碰撞生物力学是通过研究和分析人体损伤的过程，弄清其损伤机理，从而幵发出有效的方法来减少或消除人体的结构或功能的损伤。研究内容包括：碰撞损伤机理、量化人体组织的机械响应、人体组织损伤准则、研宄开发对人体保护装置、模拟人体的碰撞响应。损伤过 程见图 9-1 2, 事故的产生， 乘员身体在外界机械载荷的作用下，采用损伤控制方法，使作 用在人体上的外界载荷减小，之后乘员身体对外界产生的机械生物力学响应也就相应减小， 最后通过损伤准则、损伤耐受度来评价损伤程度。

生物力学响应是指人体承受外界载荷情况下，人体产生的机械和生物等方面的响应，具体表现为人体的位置和形状随时间的改变，以及由于机械变化而导致的人体组织器官的生物变化，例如，事故中由于方向盘撞击导致头部的运动、变形，以及胸部变形等，这是机械响应，而生物力学响应则指头晕、头痛、胸疼等。 损伤可分为两种形式；侵入损伤和非侵入损伤。侵入损伤一般有高速物体或低速尖锐物 - - 体撞击人体。而非侵入损伤一般由钝的物体撞击人体所致。由这两种损伤形式导致人体损伤一般有三种类型

  • 1、 弹性损伤 ( Elastic Injury )-->人体某部位受到挤压超过其弹性承受的极限，这种形式损伤可能由低速的身体变形或高速撞击形成的，例如，交通事故中人体的骨折等， 就属于这种损伤。
  • 2、 粘性损伤( Viscous Injury)-->人体受到突然的冲击使得人体内部器官超过其粘性极限，这种损伤，在外部没有明显的痕迹，但这种损伤依然存在，例如，胸部受到撞击往往会产生粘性损伤。
  • 惯性损伤 ( Inertial Injury)-->人体在惯性作用下，内部组织结构出现撕裂等现象， 例 如，头部在巨大的加速度情况下，脑室液经过脑室系统猛然移动，导致颅内出血等。

生物力学试验一般有四种方式:

  1. 自愿者进行试验。这种试验必须确保人体不能遭受损伤，试验前要制定严格的制度 和试验规范，所以这种试验只能提供人体在没有发生损伤情况的响应的一般知识。
  2. 尸体试验。这是一种非常贴近实际的一种试验方法，通过试验能验证机械模型和数字仿真模型，但这种试验无法验证肌肉的强度和生理反应等。同时这种试验遭受了来自伦理的诸多问题，现在基本不釆用。
  3. 机械模型。制作模拟人体的试验假人，在结构、尺寸、质量分布、 运动特征等方面与人体相似，在重要部位具有碰撞生物力学的仿真度。各种试验假人现在广泛应用与车辆的碰撞安全开发。
  4. 动物模型。用麻醉的动物代替人体进行损伤试验，取得脑组织、脊椎等部位的数据，但动物与人体在尺寸、形状和结构上差异很大，很难将动物的损伤与人体损伤关联起来。随着计算机水平的发展，人体的运动学响应和损伤可以通过计算机仿真模型来研究，但该仿真模拟的输入数据还难保证可靠性和有效性。

通过大量的生物力学试验和仿真，制造了用于评估乘员伤害的各种ATD设备，同时也制定了各种 ATD设备的损伤评估参考值（IARVs)，这是我们使用ATD设备来评估我们车辆安全设计的基础。例如对于Hybridl50th假人胸部遭受5%概率的 AIS3 伤害胸部压缩量是22mm，那么遭受同样的伤害在侧碰WorldSID50th上胸部压缩量则是 28mm。所以不同的ATD设备获得的伤害数值要根据 IARVs 来确定其伤害等级。

损伤等级定义

在实际交通事故中乘员受伤形式和程度千差万别，为了研究事故中乘员的伤害情况，需要对比和确认乘员损伤的程度,这样便于分类研宂，制定相应的标准和策略。为此，1969 年，美国医学会和美国汽车医学会召集各专科医生和有关研宂人员、专家和教授，结合他们的医疗实践经验，查阅对比了数千例不同类型的损伤资料，设定不同程度损伤级别数字，这就是简明损伤分级标准 AIS ( Abbreviated Injury Scale )，后经过两年的临床试用后，与 1971 年正式发布，后来分别在 1980 年、1985 年和 1990 年分别进行修订，一直使用至今，现在使用的是 90 年版的，当时制定只是为了机动车事故损伤定级，后来逐渐应用到其他领域， 如枪伤等。AIS 依据对生命的危害程度将损伤排序，分成 6 个等级，等级越高表明对生命的威胁越高，如图所示。

在 AIS 损伤字典中，每一个损伤用一个 7 位数字表示，例如 123456.7, 第 6 位与第 7 位之间用小数点隔开，第1 位数字表示损伤的部位，分成九个区域，例如，皮肤，头颅，颈部，胸部等；第 2 位数字为典型的解剖学分区，第 3、4 位数字表示特定的解剖学分区，第 5、6位数字为损伤严重程度代码，例如骨折、裂伤、撕裂， 第 7 位数字就是前面讲的 AIS 简明等级。对于我们所研宄的车辆碰撞安全，无论是设计还是评估，损伤等级仅限于 AIS4 及以下等级， 不研究 AIS5 及以上的等级的乘员保护，因为 AIS5 及以上乘员基本无生还可能，研究没有意义，例如，EuroNCAP 评价标准中，64km/h 偏置碰 Hyridlll50th 假人胸部压缩量为 22mm，乘员损伤等级 AIS>3的概率为 5%。

损伤机理

损伤机理是人体在外载荷作用下，组织器官响应最终达到其承受极限，而导致结构失效或者技能丧失的作用机制，前面己经说过，损伤机理分为:弹性损伤、粘性损伤和惯性损伤。本章将简要介绍碰撞安全中经常评价的人体部位的损伤机理，通过对各部位的损伤机理的了解，有助于更进一步理解碰撞安全中假人的各评价指标，理解碰撞中乘员伤害的深层次原因。

头部损伤

在汽车交通事故中，乘员头部损伤往往是头部遭受突然的外界机械载荷所致，是最常见的损伤类型之一，同时也是造成重伤或死亡的主要原因，头部的损伤带来大量的社会和经济问题，根据 1992 年的有关报告指出，在美国头部损伤在所有损伤造成的死亡中占到 68%,占损伤总数的 34%，占所有交通损伤的 50%，由此造成的直接和非直接的损失达到 250 亿美元/年。

损伤形式

  1. 颅骨骨折
  2. 脑挫裂伤
  3. 硬模外血肿
  4. 脑震荡-->重伤或需要长期治疗的损伤类型
  5. 硬膜下血肿-->重伤或需要长期治疗的损伤类型
  6. 弥漫性神经轴突伤-->重伤或需要长期治疗的损伤类型

在这些损伤中，惯性损伤占很大比例。

颅骨骨折

颅骨骨折分位凹陷性骨折和线性骨折，按照部位来分， 又可分为颅顶骨折和颅底骨折。碰撞中骨折主要和颅骨碰撞点以及接触范围有关，当颅骨碰撞区域较小时发生凹陷性骨折，这是由于碰撞点受力集中，能量通过局部骨折吸收；当碰撞区域较大时，能量扩散，容易产生线性骨折。颅底骨折较为常见，产生主要原因是颅底直接冲击和颅顶轴向冲击造成对侧的应力集中。

脑挫裂伤

脑挫裂伤是最常见的头部损伤，有同侧挫裂伤和对侧挫裂伤之分，如图 10 1 1 所示。脑挫裂伤是惯性力作用而不是接触力作用的结果。在头部受到前后冲击或者惯性作用时，由于头部惯性作用，加速度导致颅骨和硬脑膜相对于脑组织的运动，另外由于颅骨结构的不规则性，脑部受到阻碍产生的剪切力，超过极限，这样就会导致组织撕裂。当头部受到冲击小时，能量大部分在碰撞点被吸收，造成同侧挫裂伤，而受到冲击大时，能量大部分被头部运动的启动和停止所吸收，这样往往同侧挫裂伤较小，对侧挫裂伤较重。

硬膜外血肿

这种损伤主要原因是外载荷对颅骨的局部冲击，产生于颅骨内面和硬脑膜的骨膜层。一般由于折断的骨边缘造成的内面硬膜静脉或动脉撕裂，导致血肿出现。如果冲击剧烈并无骨折发生，颅骨变形造成硬膜从颅骨的内板剥离，也会造成血管撕裂，从而导致硬膜外血肿。这种损伤后，会导致短时意识丧失，有 6~8 小时清醒期，随着血肿发罩，会再次陷入昏迷。硬膜外血肿常与颅骨骨折联系在一起。

脑震荡

脑震荡是典型的惯性损伤所致，它是弥漫性脑损伤的一种较为温和的表现形式，一般长时间的角加速度引发大脑深层结构变形，发生脑震荡并引起神经功能障碍，导致失忆，丧失知觉，昏迷等。

硬膜下血肿

这种损伤危险程度很高，占脑损伤总数 3~6%，但死亡率高达 35~50%，这种损伤由碰撞接触或者惯性载荷引起的，造成脑组织相对于颅骨变形或者运动，损伤产生与硬脑模和脑组织之间，这是加速度作用的结果，导致张力开始向颅内发展， 当加速度峰值很高的情况下，可能导致桥静脉和动脉的撕裂，如果加速度峰值较低而时间较长，则会导致大脑深层的弥漫性轴突伤。

弥漫性神经轴突伤

弥漫性神经轴突伤是闭合性脑损伤中一种严重的原发性脑损伤，在交通事故中比较常见。这种损伤通常是由加速度作用引起，是由较低的角加速度持续作用导致的脑部深层的变化而导致的，这种损伤严重性取决于加速度的大小、持续时间以及起始角加速度和运动方向。是一种危险性很高的损伤， 昏迷数天或数周，半数在伤后一个月内死亡。

头部损伤

头部损伤一般发生在碰撞瞬间和之后的一段时间里，通过生物力学研究归纳损伤的机理， 损伤可能是某种机理，或某几种机理共同作用的结果。

人体头部最外层为头皮，然后是不同颅骨结构，再往里就是两层脑膜和膜下空间，最后是被血管和内脑膜包裹的脑组织，从外到里，各层结构硬度和刚度差异很大，这样头部在遭受外部作用时，导致颅骨和颅内组织的响应形式不同，最终决定头部损伤的不同类型。有三种载荷作用于头部:

  1. 直接碰撞。汽车事故中，各种周边硬物与头部碰撞，造成接触力的作用效果。比如头部撞击方向盘等。
  2. 加速度载荷。碰撞中，惯性作用下，头部相对于区别的运动，造成头部加速度的变化。
  3. 静态载荷。交通事故碰撞时间极短，这种载荷一般在碰撞中不存在。

以上三种载荷作用的结果，导致颅脑结构出现变形，头部产生不同类型的应力：压缩应力、拉伸应力和剪切应力。通过几十年的研究， 生物力学研究人员得出导致头部损伤和引起脑内组织特定应力类型的载荷类型，给出了两种损伤机理：平移加速度损伤机理和旋转加速度损伤机理。

  1. 平移加速度损伤机理。头部做曲线运动，在以某固定的坐标系下测量任意两点的矢量保持为常量，这样产生直接的线性加速度，这种加速度往往导致挫伤，这是压力波在远端作为张力波传递的结果，由于压力梯度的存在，剪切应力在脑内产生并导致脑损伤。可以这样解释：惯性加速度载荷作用在颅脑系统上， 导致头皮和颅骨的变形， 并导致颅内压力升高，出现压力梯度，由于脑与颅骨是非常松散的耦合， 这导致内容物的相对运动，从而导致脑组织变形。
  2. 旋转加速度损伤机理。当头部遭受旋转加速度时，脑组织层面间将会产生剪切应变，产生了颅脑间相对旋转位移， 这样的剪切应变将损害桥静脉，导致脑溢血和血肿,继续旋转加速度，剪切应变将扩展到脑的深处，由于脑组织有相对较低的动态剪切模量，这样导致大面积的脑组织损伤。

但后来研宄表明，颅骨的变形和头部的旋转是头部损伤的主要致伤机制，因此头部的损伤最常见的是这两种损伤的综合损伤机理的结果。

评价标准

损伤评价标准是通过相关的物理参数或一个由若干物理参数组成的函数来定义评价。与人体部位受到损伤的危险程度相关，用来衡量负载是否超过导致某种损伤的程度。相对应的，耐受度则是对于物理参数或函数的阀值，超过它就会产生某种伤害或者有可能导致某种损伤。头部伤害评价标准有很多，这里介绍我们工作中常见的HIC。

颈部损伤

人的颈部是人体最易受伤的部位之一，在汽车碰撞事故中，人体的颈部也易遭受损伤在前碰中颈部损伤占到 14%左右，而追尾事故中颈部损伤占到了 33%左右，而 90%的颈部损伤是发生在低速追尾碰撞中， 且大多数属于轻微 AIS1 级别，但颈部损伤往往伴随头痛、颈痛、神经痛等症状，需要长期治疗，使患者承受巨大痛苦。
颈部损伤有闭合性损伤和开放性损伤两种， 而我们交通事故中产生的颈部损伤基本是开放性损伤，交通事故中颈部伤害以挥鞭伤（Whiplash Injury) 为主。挥鞭伤被认为是追尾碰撞头部伤害的主要原因，但还缺乏医学上关于其成因的病理解释。

挥鞭伤（Whiplash）

挥鞭伤其定义为由后方或侧方撞击所致的颈部加速减速机制所造成的骨或软组织损 伤，如椎体上软骨损伤，棘突间韧带损伤等。软组织特点是柔软易变形，具有不同程度的抗拉强度，但不能抗弯和抗压。
挥鞭伤最初由Crowe于1928 年提出，专指机动车追尾撞车对乘员头颈产生突然的过伸及过屈作用力，使颈部软组织承受过度应力而造成的损伤。在人体整条脊椎（Vertebrae ) 中，以颈椎（Cervical Vertebrae ) 的活动范围最大，颈椎的活动方式可以分为侧向摆动、左右转动和前向弯曲与后向伸张。人体颈椎分成七块，由上到下依次为第一颈椎 (C1) 至第七颈椎（C7) ， 第七颈椎下接第一胸椎（T1 ) ，除了第一颈椎C1与第二颈椎C2之间，其余颈椎间都有椎间盘，在正常情况下，椎间盘如同减震器一样吸收颈部在活动中产生的压力，保证颈椎不受伤害，但在过度用力或突如其来的外力太大时，颈部将受到伤害。在发生追尾后碰撞时，一般而言颈椎会因低头与抬头而跟着前后弯曲，在人体承受范围内的运动是不会造成伤害的， 但后碰往往导致颈椎出现不符合生理的运动，从而产生伤害。当车辆被追尾时会加速向前，同时汽车座椅施压推动乘员的躯干往前运动，头部却因惯性而停留在原来位置，产生头部延迟效应，由于躯干与头部明显地朝相反方向移动，颈椎的上半部分因头部向后伸张，而颈椎的下半部分因躯干被向前拉伸， 此时颈椎呈现 “S” 型变化，接着颈部还要承受头部向后及向下的压力导致的颈椎过度伸张，并出现 “C” 型变化。然后头部受到座椅头枕的反作用力又迅速前摆，此时颈部要承受向前及向下的压力导致颈椎过度弯曲，产生伤害，这个过程如同挥鞭甩动一样，故名挥鞭，这种损伤也叫挥鞭伤。

在颈部挥鞭过程中，头部和胸部有较大的相对运动，使颈部过度后伸，超过了人体运动的正常生理范围，造成颈部软组织拉伤和扭伤，轻微的AIS1损伤有下面三种:

  • 椎体及周围韧带或肌肉损伤。椎体上下面的椎骨板， 有一部分是粘弹性材料，它的刚度比椎骨其他部分低，在同等情况下，椎骨板比椎间盘先损坏，在追尾碰撞瞬间，椎间盘相对于椎体的运动很剧烈， 对椎体上下钩突和前后的肌肉或軔带产生一个很大的剪切力，导致椎体周围肌肉或韧带的撕裂，或者导致交感神经或椎体上钩突部位表面的损伤。
  • 脊髓神经根损伤。在碰撞瞬间，在速度和加速度作用下，脊管内体积瞬时发生很大变化，引起脊髓的压强瞬时变化，可能导致脊髓神经根损伤，它是颈部损伤NIC准则的基础。
  • 关节突部位损伤。通过试验发现在颈椎运动的瞬间，几块颈椎旋转中心偏上，并且C6比其他颈椎开始伸展的时间早一些， 这时形成S型， 其他颈椎随着C6伸展，在 C5和C6之间产生一个极大的转矩，使脊柱前纵韧带拉伸，以及C5的下关节突关节面与C6的上关节面发生冲撞，导致关节突上软骨面和周围关节囊的拉伤。

软组织不能承受压载， 压载太厉害，变形太大，就会导致软组织的扭伤或拉伤，并且 在运动不正常的情况下，除了压缩力外，它还可能受到剪切力，扭矩等。 挥鞭伤的并发症极多， 据统计，临床表现第一是颈脑综合症， 其特征是头痛、疲劳、眩晕、注意力不集中、调节障碍以及对光的适应能力减弱等。第二个综合症也叫下颈段综合症， 其特征是颈痛和颈肩痛。其它的症状还可能产生吞咽困难、 头晕、视力障碍、颅祌经损伤、自主神经系统损害、颞下颂关节功能障碍以及斜颈、前胸痛等。颈部挥鞭伤一般 不会造成生命危险，比较常见，但治疗费用高，还可能产生长期、明显的后遗症。如果未 及时治疗极有可能导致慢性不愈，有时两者可同时发生。由此我们可以看出，挥鞭伤虽没 有生命危险，但会给受害者带来很大的痛苦，也会给社会带来巨大的经济损失。

人体的颈部自由度很大，可以弯曲和转动，因此颈部可能受到外界好多种载荷，任何一种运动方式都可能导致颈部损伤。颈部可能受到的外界载荷：弯曲 ( Bending)、压缩（Compression )、拉伸（Tension )、扭转 ( Torque ) 和剪切 ( Shear), 在车辆碰撞安全中，颈部基本受到：弯曲、 压缩、拉伸和剪切。

无论在前碰中，还是在追尾后碰中，弯曲是一个很重要的考察指标，颈部弯曲分成两种形式:弯曲(Flexion) 和伸张 ( Extension )。但在实际中，颈部往往是两种载荷的组合在一起对颈部的损伤，如图 10-2-5 所示，最常见的是：压缩一弯曲（CF ), 拉伸一伸张（TE)。另外还有：压缩一伸张（CE), 拉伸-弯曲（TE)。在一次碰撞中，颈部的损伤过程基本是这几个的组合过程。在 NCAP 前碰颈部评估指标“弯矩”是指的上面TE 组合，所以弯矩值取ly曲线的负向最大值。

胸部损伤

在交通事故中乘员胸部伤害， 胸部伤害是交通事故中导致乘员重伤和死亡的主要原因之一。根据美国国家事故釆集系统（ NASS ) 统计，交通事故中，胸部损伤占 267%。而根据欧洲相关事故统计数据，在欧洲交通事故死亡人员中胸部损伤所占的比例约为 33%。在事故损伤中胸部损伤占所有轻微或中等伤害 (AIS2) 的比例为 13%, 占严重或致死伤害 (AIS3) 的比例则为 29%。因此，乘员的胸部保护是汽车安全性设计的重要内容之一。

损伤机理

在车辆发生碰撞时，人体的胸部承受各种约束作用，与车辆内饰各部件发生作用，例如，驾驶员胸部与方向盘作用，胸部与仪表板表面作用，胸部与安全气囊作用，胸部被安全带勒紧，侧碰时胸部被车门撞击等，在这些作用过程中，将胸部的产生的生物力学响应分成三类:

  • 加速度对躯干的惯性的阻力
  • 骨骼结构和软组织的压缩变形
  • 软组织的粘性响应。

胸部损伤主要有肋骨骨折和内部器官等软组织损伤两种。上面说过当胸部压缩过大，将直接导致肋骨骨折和胸腔内器官受损。即使胸部压缩量在讨接受范围内（不发生肋骨骨折),但也会因为胸部压缩速度太快（粘性载荷）也很可能导致胸腔内部软组织器官受损，如肺部挫伤。另外，胸腔内部器官常常遭受加速度载荷的惯性冲击，引起器官软组织的剪切或撕裂，如心脏和大动脉撕裂。在实际交通事故的碰撞过程中，乘员胸部损伤机理常常是由以上三种任意组合造成的。
肋骨是人体胸部的主要组成结构，起着承受大部分冲击载荷，保护内部器官的作用。由于肋骨的近似椭圆结构及内部器官的非对称布置，胸部在冲击载荷下的生物力学响应及损伤情况和加载方向密切相关。相关研宂表明，在相同沖击能量下，尽管胸部侧面冲击产生的胸部侧向压缩量比正而冲击胸部正向压缩量要小，但却导致更多肋骨骨折 。这是因为肋骨侧面曲率更大，且表层覆盖的肌肉组织较薄，纯正向和侧向冲击相比，斜向冲击很可能导致更严重的乘员胸部伤害，这在侧面柱撞工况中得到了证实。尽管钝形冲击下的胸部损伤机理适用于各方向冲击工况，但在不同方向的钝形冲击下，胸部的损伤情况截然不同。尤其当胸部左前侧遭受斜向冲击载荷时，将很可能发生心脏挫伤和大动脉破裂等软组织损伤，对乘员生命造成极大威胁。经过统计，胸部损伤导致死亡的往往是胸部肋骨骨折，产生的尖锐棱角刺穿大动脉导致失血过多而死亡。所以 评估中，胸部的安全设计重点是降低胸部压缩量指标。

胸部损伤准则

现有的胸部损伤准则主要分为单一变量准则和复合变量准则。单一变量准则如加速度准则，粘性指标 VC ( Viscous Criteria ) 和胸部压缩量。每个单变量损伤准则对应相应的胸部损伤机制。复合变量准则有复合加速度准则 TTI (Thorax Trauma Index ), 复合胸部损伤指标 CTI ( Combined Thoracic Index )。

腹部损伤

在交通事故中，腹部是人体最容易受伤的部位之一，常发生腹部的钝性撞击损伤。虽然在损伤之初并不明显，但可能有生命威胁。例如，在侧面碰撞造成的严重损伤（ AIS54 )中,有 20%是腹部损伤。因交通事故伤及腹腔主要脏器致死的占交通之故总死亡率的14%~24% 0据报道，腹部损伤最常见的是实质性脏器损伤，如肝脏、脾脏、肾脏等，主要是由汽车发生碰撞时前部转向系统、侧门、仪表盘、安全气囊等。

损伤机理

车祸时，腹部常受到瞬间猛力的撞击，而导致腹腔内脏器的损伤。此类损伤以闭合性腹伤为主，且同时累及肝、脾、肾等重要的腹内脏器。腹部的解剖学结构较为复杂，故存在很多因素影响碰撞严重程度。

首先，腹部骨结构对腹部内脏的保护较少，内部器官在前部和侧面撞击中更容易产生变 形而造成血管破裂与内部组织的损伤。

其次，腹部脏器的解剖学、组织学特征对其损伤风险有直接影响。实心的器官比中空的器官更容易受到损伤。腹部空腔脏器如胃、小肠、大肠等损伤程度一般为AIS1~2, 而其由他实质性脏器如肝脏、脾脏等损伤引起的伤害则上升至AIS3~6。肝脏位于右上腹，紧靠腹壁，是腹腔中最大的实质性脏器，且质地脆，相对固定。上腹部的钝器冲击使肝与脾的移动生成显著的变形，而肝脏压缩会增加肝内压力，产生拉伸或剪切应变。车祸时强大的撞击力作用会导致相对位移和不均匀性变形，致肝脏发生撕裂性或破碎性损伤。如果组织变形较大将撕裂肝血管，导致腹腔积血。腹部变形导致肝的相对运动也使血管产生拉伸和剪切。腹部器官非对称组织形式也使得不同的损伤风险依赖于碰撞部位。

再者，在汽车碰撞屮，汽车内部的转向装置、侧门、扶手、安全带、安全气囊等对损伤结果影响很大。比较常见的碰撞有：驾驶员腹部与方向盘发生碰撞；前碰中乘员下潜安全带腰带滑到腹部，给腹部的勒力；安全气囊引爆对腹部的作用等。来自于1993~1997 年间美国国家事故采集系统 NASS的数据显示，由驾驶舱侧面包括内饰、扶手在内的装置引起的AIS3~6碰撞损伤的比例高于由驾驶舱前面包括挡风玻璃、转向装置、仪表盘、杂物箱在内的装置引起的同等程度损伤的比例。驾驶员的肝脏破裂则常为方向盘的直接作用所致。在国外，方向盘的变形量被建议作为评定胸部损伤、腹部损伤和整体损伤程度的预测值运用到汽车碰撞事故中。
钝性碰撞所致腹部损伤的严重程度不仅与腹部的解剖学结构、撞击物特征、碰撞位置有关，还取决于撞击能量传递形式，撞击载荷的质量、速度和作用方式等因素。动能的传递形式与创伤模式和严重程度存在密切的联系。
Viano等人指出，机体受到物体撞击吋，致伤的危险性与冲击物体传递到机体的能量有 关。研宄者们通过尸检对 293例平面跌落伤及台阶跌落伤分析发现，仅 4 例存在肝脏破裂，其中 3例为腹部受到坚硬障碍撞击计算发现，跌落时的能量传递时间较长，对肝脏影响相对较小。撞击速度是影响腹部创伤程度的关键因素。 当速度分别为 4~7m/s时腹部损伤情况，结果出现 2 个 AIS4、1 个 AIS3 和 3 个AlSO. Rouhana 等指出当速度超过 5m/s,只考虑侵入量己经不足以描述腹部损伤程度，还要把速度因素考虑在内。 Lau等人进行腹部撞击试验中保持压缩率为 16%不变， 得出速度高于12km/h 会伴有肝 损伤，速度髙于 14km/h 会引起肾损伤。Cavanaugh 等于 1986年报道了正面撞击中掩击速度与腹部刚度的关系，指出刚度与撞击速度和撞击质量成比例关系。 Viano等进行腹部侧面撞击时指出高速状态下如速度达到 9.8m/s时出现肝撕裂伤和肋 骨骨折，损伤程度为 AIS3~4,此时粘性损伤准则己不足以评估损伤程度。

器官压强是经常用来评定组织损伤程度的参数之一。胸部和上腹部撞击发生肺、肝及大血管损伤时，胸、腹内压都升到很高的水平。但对于有的实验动物，胸腹内压虽升到很到水平， 也不发生肺、肝严重损伤。说明胸腹撞击时，胸腹内压升高只是引起胸腹腔内脏损伤的因素之一。 Hardy等研宄正面撞击效应时发现肝脏内压强随着撞击速度的增加而增大，撞击速度达到 6m/s 时压强可以达到 167kPa，即可出现肝脏前缘纵向撕裂伤。撞击速度达到 9 m/s 时压强达到 240 kPa，同样出现肝脏撕裂伤。 Walfisch等进行的尸体坠落实验得到 260kPa 的脏内压能引起 AIS>3的肝损伤。

腹部伤害评价准则

部损伤与胸部损伤评估有很多相似之处，腹部评估指标如下。

  1. 力。Walfisch 等人探究腹部侧面撞击损伤时发现腹部受到 AIS3+损伤时，最大力为4.5kN, 最大压强 260kPa。
  2. V*C 粘性指标。Rouhana 等人证明 V*Cmax 更能体现腹部损伤情况，更适合作为腹部损伤预测参数。在速度很低时， 最大压缩量可以预测腹部损伤情况，但随时速度增大，超过 5m/s 时，仅用压缩量己经不合适了，还应该把速度考虑在内，速度与最大压缩量的乘积要比单独的最大压缩量或最大速度更能准确预测损伤风险。Viano 计算出 50%概率 AIS4+损伤 V* C=2.26m/s, 压缩量 C=46.8%, 而对于 50°/。概率AIS+损伤 V*C=2.01m/s，压缩量C=51.2%。

髋部损伤

髋部损伤主要来自车辆的侧面碰撞中，在侧碰中，乘员的髋部骨盆容易受到损伤，根据美国国家事故釆样系统 NASS 统计表明，所有侧碰事故占整个事故的 15% 30%, 侧面碰撞乘员身体各部位的损伤中骨盆占 8°/。~14%, 侧碰中乘员离车门距离很短，撞击的能量不易被吸收，髋部骨盆极易受到伤害，骨盆损伤的类型按照发生的频率依次为：耻骨骨折 50%，股骨骨折 25%，髋S骨折 12%，骶髂关键骨折及骶骨骨折 11%，

骨盆损伤通常由于侧向冲击产生的，由冲击压缩力形成的骨盆损伤机理占主导地位。横向冲击作用在髋部骨盆上的侧向载荷导致股骨压缩损伤，损伤部位往往涉及一个或多个部位：耻骨联合、耻骨支、髋臼、股骨和股骨近端。通过生物力学尸体试验，对 50th 的男性成人的骨盆进行侧向冲击，骨盆耐受限度均值为10kN,而对 5th 女性成人骨盆则为4KN.

下肢损伤

依据 NASS 数据库，对 1994 2007 年间的前碰进行统计，驾驶员下肢也最易受到损伤,占所有车内乘员下肢损伤总数的 82.5%。而对 1993 2007 年间的在安全带和安全气囊保护下的前碰乘员损伤情况：55%的下肢损伤部位位于膝关节 大腿 髋关节（KTH Knee Thigh Hip )，同时下肢损伤占到所有 AIS2+损伤的 36%,超过了头、胸部位。下肢损伤尽管不会危及生命,但致残率很高，康复周期长，给伤者和社会带来沉重负担。

损伤机理

乘员的下肢损伤基本来之前碰的乘员，在前碰中乘员受到车辆的纵向向前的减速度，在约束系统作用下，乘员的下肢与车辆内饰接触，这样产生了下肢的损伤，这些损伤来自乘员下肢受到内饰部件的碰撞和挤压，人体的下肢运动范围大，而可能与下肢接触的车内部件较多，这样乘员的下肢动态响应非常复杂，有很大不确定性。
人体的下肢有三个主要的组成部分：大腿、小腿胫骨和脚，同时还有连接这三部分且活动自由度很大的关节：膝关节和踝关节。脚由 27 块骨骼组成，骨骼间的活动范围没有膝和脚踝那么大。下肢的损伤机械响应一般是骨骼的弯曲导致的骨骼拉伸，这种情况在大腿和小腿的长骨很常见，这种损伤最终往往导致骨折损伤。还有一种常见的损伤是关节靭带的软组织损伤，例如，腱撕裂，半月板破裂，软骨损伤等。
车辆碰撞时， 乘员下肢的损伤是与车内众多部件接触的结果，Huelke 对 NCSS 数据进行统计分析，碰撞中乘员下肢与车内发生接触的有仪表板、脚踏板、方向盘、转向管柱、中控台、前排座椅靠背等。对于驾驶员，与仪表板的接触是引起膝部、大腿和髋部骨折损伤的主要原因，而 53%胫骨和腓骨损伤归因于仪表板，37%归因于前围板和踏板， 前围板和踏板也是引起脚及脚踝损伤的主要来源。
为了便于研宂， 般将下肢分成两部分：膝-大腿-髋部 ( KTH ) 和膝下部分。

乘员保护分析

头部保护分析

头部是人体最容易受到伤害的部位之一，而且也是最容易致命的部位，所以乘员保护的重点是头部的保护，随着乘员保护的技术发展，到现在对头部的保护相对比较完善，因此在现在事故中，头部遭受严重伤害的概率在降低。在现在 NCAP 评估中，那些星级较高的车型，头部评估基本是满分。在 NCAP 评估中，通过物理假人来评估乘员头部受到的伤害情况，一般通过假人釆集的数据不能完全反映头部受保护的性能，因此 NCAP 引进了很多额外的评价标准，即虽然通过假人釆集数据头部获得满分，但头部保护存在潜在风险，依然要被扣分。这些评估主要是针对气囊的，因为气囊是把双刃剑，用的好能保护乘员，用的不好反而会伤害乘员。

颈部保护分析

法规对颈部的评估一般是在前碰和后碰中，一般在这种工况下，颈部容易受到损伤。对于前碰来说，对于前排乘员，由于在安全气囊和安全带综合作用下，颈部不容易受到伤害，因此在 NCAP 评估中，基本能获得满分的评估， 但如果没有气囊的辅助约束，则乘员的颈部极易受到伤害，例如，没有配备气囊的前排或者后排乘员。
前碰中乘员颈部受伤害是因为头部在惯性作用下向前运动，而躯干被安全带约束，导致颈部受到过大力和弯矩。对于前排乘员，有时配备了安全气囊，颈部也有可能得分低， 这往往发生在副驾驶侧,由于副驾驶侧，空间较大，如果气囊布置较小，或展幵较早，那么气囊没能很好的托住乘员的头部，导致头部在被气囊托住之前，向前运动幅度过大。同时头部相对于胸部向前运动幅度过大，也会加大胸部的伤害。

在前碰中， 假人受力示意图如所示， 当安全带限力起作用时，将假人的躯干拉住，限制了假人躯干向前运动，在这过程中，头部在惯性作用下的运动趋势是平动，而躯干的运动趋势是绕髋部前倾，这样的，颈部对头部有向后，向下的力，相反地，头部对颈部有向前向上的力，因此，在头部与气囊接触前，上颈部剪切力 Fx 是向后的（负），轴向力 Fz是拉伸力（正），由于颈部有负的剪切力 Fx，因此颈部存在伸张弯矩Mocy。

从伤害看，伸张弯矩Mocy超过42Nm ，这是头部向前惯性大，头部没有受到气囊有效约束所致，同时头部通过颈部拉动躯干，也导致胸压较大。因此，可以得出结论，气囊排气孔过大，泄气过快，导致气囊压力不够，对假人头部向前运动约束不够。将气囊的排气孔调小，减小泄气的速度，增加气袋压力，从而增强气囊对头部运动的约束，这样颈部伸张弯矩Mocy和胸部压缩量都有明显的降低。从生物力学的伤害机理来说，尽量减少头和胸的相对运动，就能降低颈部的伤害，后碰的挥鞭伤也是一样，比如主动式头枕，其保护原理也是这样，主动托住乘员头部，减少头相对于胸的运动

胸部保护分析

胸部是乘员保护性能开发的重点， 胸部容易失分，而且胸部的伤害与很多因素有直接的关系，例如，气囊、安全带、座椅、IP，乘员的运动姿态等都会影响胸部的性能。因此胸部保护性能的分析和开发都比较复杂。

这里着重阐述正碰乘员的胸部分析

在前碰中，驾驶员大致受力情况大致如图所示，可以看出，约束系统对乘员的保护是通过这些力共同作用的结果，从约束系统匹配角度，希望乘员所受的力是相容的，相互均衡的，也就是，不是希望某个力越小越好，也不希望某个力越大越好，而是各力相互均衡，共同作用于乘员。对于乘员胸部，如果气囊对乘员胸部力越小，那么安全带对乘员胸部力将越大，因此胸部伤害将增大，反之也如此。所以在前碰中对乘员胸部的保护关键使各作用于胸部的力相互均衡， 这样才能达到最佳的保护效果。

气囊安全带因素分析

在前碰中气囊可以托住胸部，对胸部起到缓冲作用，所以气囊的位置和内部压力对胸部伤害有很多影响。例如，有的 DAB 采用“三点和九点”两根拉带，当乘员胸部扑向气囊时，挤压气囊，气囊容易上翻，导致气囊与胸部接触的不稳定，这样不利于气囊对胸部的保护。无论是全正碰还是偏置碰，气囊点火时间相对早点比较好，这样气囊可以较早参与对胸部动能的缓冲吸收，但这时要保证气囊内部的压力，如果泄气太快，那么到后期胸部运动剧烈时，气囊不能起到很好的缓冲吸能作用。所以气囊的泄气快慢、气囊点火时间需要通过滑台试验进行匹配。

在对乘员胸部保护中，安全带起到主要作用，现在很多前排安全带系统都配备了预紧和限力功能，这些功能参数和气囊需要通过滑台试验进行匹配。由于假人本身的特点，因此安全带的佩带位置对假人伤害有较大的影响，所以在前期布置和后期的试验中，必须确认安全带对假人的缠绕位置。必要时，可以将安全带高调器设置在最高位置，有时这种方法效果很明显，可能对偏置碰副驾不太明显。

安全带的肩带力大小虽然影响乘员胸部保护性能，但肩带力也不能降的太低，有时候导致胸部压缩量大往往不是肩带力，而是由于下肩带力过大，即 B4 力。可以分析一下假人胸部 Z 向加速度，可以给分析乘员胸部伤害原因提供参考。
就以偏置碰工况为例。安全带的肩带力在胸部产生的 Z 向加速度是负的，而 B4 力一般产生正的 Z 向加速度， 如图所示，驾驶员侧胸部 Z 向加速度和胸部压缩量曲线，从曲线看出，当 Z 向加速度达到正向最大时，胸部压缩量也达到最大，那么胸部压缩量过大与安全带 B4 力过高有一定的关系。

而乘员侧，在前碰过程中，乘员的运动空间相对较大， 乘员躯干在向前倾斜过程中，B3力和B4 交替增加，如图，往往是胸部 Z 向加速度达到正向最大时迅速下降，达到负向最大，这是由于正向最大时刻腰带力迅速卸载，而肩带力依然保持限力的水平， 所以胸部 Z 向加速度迅速反转。

转向管柱压溃也用于对胸部的保护，但在滑台试验中很难去调整管柱的压溃性能参数，所以在前期根据经验或者通过 CAE 手段确定管柱的压溃参数。

另外在前碰中要确保方向盘不能碰到胸部或者腹部，或者说，气囊不能被胸部击穿。这种情况在 EuroNCAP 要被扣分。胸部与方向盘接触要通过多方面来判断，通过高速摄像，确定假人胸部沉入气囊的时刻， 然后通过胸部加速度曲线来看看有没有瞬间增加的加速度峰值，再考察这时的安全带力有没有对应的变化，如果安全带力有对应的变化，那么这个胸部加速度瞬时峰值可能是安全带所致，如果安全带力平缓，那么可能是胸部碰到方向盘。试验后也需耍查看假人 T 恤衫等有没有接触的痕迹。不能通过胸部压缩量曲线来判断胸部是否与方向盘接触。

例如图 14 3 4 所示的胸部加速度曲线，在 103ms 左右有一个加速度峰值这个时刻气囊的内部压力随着泄气己经很低了，通过查看安全带的力，在此时刻没有摄动，试验后查看了方向盘，在方向盘下轮缘并没有永久性的变形，因此可以判断在 99~105ms 期间，方向盘碰到了假人的胸部。

座椅因素分析

座椅性能影响乘员胸部的保护，这个往往被忽视。众所周知，在前碰中乘员下潜，对乘员的伤害很大，这其中就有对胸部的伤害。当乘员下潜时，安全带勒住乘员胸部，同时气囊对乘员胸部也保护不足。

验后重点考察座椅的变形情况， 记录其变形情况结合座椅变形情况对胸部伤害进行综合的分析。

另外坐垫表面的材料对胸部伤害也有很多的影响，如果采用真皮，那么表面摩擦系数较小，前碰时对乘员的髋部约束较小，而如果釆用织布，摩擦系数较大，前碰时对乘员髋部约束较大，例如大众 Tiguan 在 2016 获得 EuroNCAP 评估五星，座椅坐垫表面就是采用织布材料。

座椅对乘员胸部的保护性能就看座椅在前碰中是否有效地约束乘员髋部。所以座椅防下潜性能、坐垫表面材料、坐垫的倾角都会影响座椅对髋部的约束。有时为了提高乘员的胸部性能，试验时可以将坐垫倾角调至最大， 这样有时会收到很好的效果。

某车型开始坐垫倾角处于中间位置经过试验后，胸部压缩量偏高，后来将坐垫倾角调至最高位置，通过试验验证，乘员的胸部压缩量降低了很多，如图，为这两次坐垫倾角带来的乘员胸部压缩量对比，仅通过改变坐垫的倾角，胸部伤害得到非常明显的改善。

有的车型坐垫在设计时考虑了减轻驾乘者的疲劳，将坐垫泡沬做的很硬，较硬的坐垫泡沫，使得乘员的髋部受到座椅骨架约束减小，再加上真皮座椅，因此在前碰中，乘员髋部前移加大，这样髋部拉动腰带，这种情况乘员腰带力会变大， 有时能达到 10kN 以上，由于腰带与肩带是通过锁扣连通的，因此当腰带力较大时，会传递给肩带下端，导致肩带下端的力变大（即安全带B4 力）， 而 B4 力直接作用于胸部，这样胸部的压缩量会增大。因此在试验时，或者仿真时可以测量安全带的 B4 力，如果 B4 力较大说明，髋部可能有较大的前移，或者说较大的腰带力通过锁扣传递给肩带。如图所示， 是在试验中通过张力器测量安全带的 B4 力，通过曲线可以看出，开始安全带 B4 力与 B3 力相当， 但随着腰带力 B6 进一步增大， 传递给了安全带 B4 段， B4 力增加很快，所以这种情况下，导致胸压升高的很大程度是由于 B4 所致。

出现这种情况，可以通过增大坐垫倾角来限制髋部的运动，还有一种方法就釆用可锁止锁舌，切断腰带向肩带传递载荷，即切断 B5 向 B4 传递载荷， 这种方法往往会收到很好的效果，B4 力会明显降下来，其至低于B3 力。

大腿因素分析

如果把人体看着一个整体，IP 给膝部的力，会通过大腿传递到髋部，势必会缓解人体的其他地方的载荷，比如胸部载荷，这样就降低了胸部压缩量。另外，大腿受力大，很好地约束了假人的髓部向前运动，这样人体在向前运动中就有很好地运动姿态，上躯干前倾，安全带对胸部的着力点上移，减轻了胸部的伤害。
交通事故中，乘员的下肢伤害大多数是 AIS2 及以下的伤害，一般不会致命，而胸部伤害往往致命，从现实意义上讲，可以牺牲下肢伤害，提供胸部的保护。

所以有的 IP 设计标准要求 IP 离膝部的距离不能太大。

当大腿的伤害容量较大（大腿轴向压缩力累积不超过 3.8jN 就不会被扣分）， 一般在 NCAP 评估中，大腿不会被扣分。因此，在前碰中，如果大腿伤害余量充足，可以考虑用大腿力来平衡胸部伤害，比如在 IP 上加装防撞横梁，或者布置 KAB 等，这些措施都可以减轻胸部伤害。

下图是胸部伤害曲线和大腿力曲线，当胸部压缩量达到第一个峰值时，这时，大腿力正急剧增加，然后人体上躯干在大腿力的作用前倾，胸部压缩量开始缓慢降低。

小腿因素分析

在偏置碰中，乘员的两脚保持在踏板上，膝部与 IP 正常接触，当髋部向前运动到极限时，假人的下肢对下躯体有个支撑作用，这时假人的髋部需要继续释放能量，可能会上翘,当髋部上翘时，安全带的 B4、B5 和 B6 力都会增加，同时胸部 Z 向加速度也会增加。如图所示。髋部上翘时，安全带幵始将乘员下拉，约束乘员髋部上翘，这时安全带 B4 和 B6 力开始增加，随着 B4 力增加， 假人的胸部 Z 向加速度也跟着增加。由于 B4 力作用于假人的胸部，因此假人的胸部压缩量也幵始上升，直到胸部 Z 向加速度达到峰值时停止。

头部因素分析

如图所示，胸部压缩量有两个峰值，前一个与安全带限力预紧有关，这时假人没有与气囊接触，第二个峰值是因为头部向前运动，通过颈部拉动胸部向前，导致的一个峰值。这个可以通过颈部的伸张弯矩看出来，正好在这一时刻，颈部伸张弯矩达到最大。另外通过视频也能看出，由于 PAB 气囊较小，不能很好地''托住"假人的头部。当第二个峰值很高时,直接影响胸部得分， 因此为了减小第二个峰值，需要增大气囊的体积。

同样对于后排的乘员，头部的运动对颈部伤害影响特别明显。后排女性假人不像前排，有更多地约束（比如，气囊， 方向盘等），后排女性假人只有安全带，因此前碰中，假人的头部是自由的，当头部向前甩动较大时，就会通过颈部拉动胸部，因此胸部有一部分伤害是由头部fe动造成的。

后排乘员胸部保护分析

与前排不同，后排乘员一般没有气囊的保护，一般来说没有比较好的方法，基本是通过安全带和座椅来对乘员实施保护。对后排乘员保护主要从以下几个方面着手:

  1. 良好的座椅设计，后排座椅最好有防下潜结构，在前碰中能有效约束乘员的髋部运动，防下潜结构不能过强，但也不能太弱，最好通过 CAE 手段优化其强度。坐垫的倾角适当大一点。
  2. 合理的布置安全带固定点的位置，确保碰撞中安全带不能出现滑肩现象。后排安全带一般没有高调器，当将安全带上固定布置太靠外时，对于 5%的女性假人来说，容易滑肩。
  3. 后排安全带系统尽可能带有预紧和限力功能，后排安全带没有预紧和限力功能，假人很难获得高的得分。后排安全带的限力不能过小，如果过小的限力，乘员头部前移量会变大，不但会被额外扣分，也会碰到前排座椅；但限力也不能过大， 过大的限力不利于乘员胸部保护，如图所示，建议在前碰中， 后排乘员向前运动到极限位置时，乘员的躯干与水平的夹角 a 最好小于 90 °。
  4. 后排下固定点锁扣最好釆用刚性，锁扣不宜过长，过长的锁扣，尤其是织带型，锁头容易被拉到乘员腹部，导致胸部安全带向下偏移，对胸部保护不利。

在正碰中，安全带肩带从乘员肩部滑落，或有滑落的趋势，如图。这时安 全带肩带就失去了肩部这个力的作用点,安全带肩带基本作用在胸部，导致胸部压缩量过大。这两个对比，安全带滑肩使胸部压缩量增加了5mm。

安全带滑肩一般出现在后排女性假人上，女性假人坐姿低，座椅前后不好调节，安全带上固定点很难进行合理布置。另外，在试验准备时，需要确认安全带对假人胸部的缠绕，以避免由于人为因素导致的安全带滑肩问题。安全带滑肩可以在车辆幵发前期总布置时需要考虑的一个问题，可以根据总布置的安全带固定点位置，通过专用软件来评估安全带滑肩风险，或者用相关 CAE 软件来仿真分析。

在 EuroNCAP 评价章程中，对成人乘员的肩带力（B3 力）有相应的修正项，即肩带力超过 6KN 要罚 2 分。如图所示，是某车型的的肩带了曲线，从图中可以看出，乘员的肩带力接近 7KN,依据 EuroNCAP 评价章程的字面意思，要进行 2 分的修正。但在实际进行评估时，并没有这么简单。首先对肩带力曲线进行 60Hz 的滤波，如图中红色虚线所示，然后考察肩带力曲线有没有连续 20ms 的区间超过 6KN 的，如果有，则要进行 2 分的修正，如果没有则不需要修正。因此我们再来看看下面的肩带力曲线，虽然滤波后，峰值依然超过 6KN，但是超过 6KN 的持续时间约 5ms 左右， 远不足 20ms，因此下面的肩带力不需要修正。

下肢保护分析

THOR假人下潜

在2020年 EuroNCAP 将实施的 MPDB 评估，对 TH0R 进行下潜修正，所以在今后中前排乘员防下潜将是设计的重点。

首先，来剖析一下乘员下潜的原因， 导致了乘员下潜的原因可以从三个方面（参数）来解释

  1. 乘员的初始位置。通过试验研究，发现乘员的骨盆初始向后倾斜的程度越大，乘员发生下潜的风险就越大！例如乘员的 H 点位置向坐垫前方移动 Xmm 的距离，那么乘员的初始骨盆角将增大。如图所示，圆圈内为骨盆的位置，不同的位置带来不同的下潜风险
  2. 安全带腰带与乘员之间的相对位置，即安全带腰带固定点的位置。国外专家通过试验指出：安全带腰带初始角度最好垂直，这样可以大大降低乘员下潜的风险。因此腰带的方向或腰带固定点的位置影响乘员下潜，如果腰带固定点太靠后，将使腰带更倾向于水平，从而增加乘员下潜的风险。在前期进行约束系统布置时，尽可能增大腰带的初始角度。
  3. 第三个原因是安全带松弛量。无疑安全带松弛量将增大下潜的风险。因此安全带系统的类型，尤其是否配置预紧器，这是降低下潜风险的关键因素。带有预紧的安全带装置减少了乘员和安全带之间的松弛，从而使安全带能有效地、正确地约束乘员的髋部，降低了下潜的风险。

最终消除了乘员下潜风险。

通过匹配相应的安全带预紧和限力，乘员的髋部得到了有效的约束，这从乘员髋部和座椅横梁的移动可以看出，降低了很多；乘员未下潜，作用在髋部的腰带使髋部的加速度上升很多，但由于限力的作用，髂骨并未出现骨折。在越来越严格的 NCAP下潜评估中，安全带系统预紧限力功能的匹配也是解决乘员下潜的一条路径。

但在实际试验中（PMHS试验），过大的腰带力，即使在没有下潜的情况下，有时也会导致髂前上棘骨折，在有下潜的情况下，髂前上棘骨折风险更大。但这一方面的损伤并没有出现现有的NCAP 评价中。

大腿

如图所示，为典型的大腿压缩力曲线，大腿由于惯性，有些时刻的力为正，即大腿处于拉伸状态，当大腿碰到IP等内饰时（如图中的圆圈的时刻），大腿力开始向负方向发展，即大腿处于压缩状态。

分析大腿伤害时，一定要结合接触开关信号，IP 上的接触漆印来综合判断膝部与 IP 的接触情况，重点要搞清楚：膝部何时与 IP 接触？接触到 IP 的什么地方？膝部有没有在 IP 表面有接触滑动？这些都是分析大腿及小腿伤害的基本信息。另外评估大腿伤害情况是考察大腿压缩力在时间域上的累积伤害！

膝部滑移量一般为负值，从前面的生物力学的损伤机理可以清楚，如果正常情况下，仅膝部与 IP 接触，那么膝部滑移量一般会很小，但如果小腿胫骨直接与 IP 接触， 那么膝部滑移量有可能很大。分析膝部滑移量的损伤情况，要结合小腿的受力情况，小腿胫骨与 IP 的接触情况来综合判断分析。

小腿

如图所示，为小腿胫骨轴向力，一般小腿胫骨轴向力有两个峰值，如图中的圆圈所示，第一个峰值是由于下肢在惯性作用下前移，受到了撞击而产生的胫骨压缩力，第二个峰值一般是在膝部与 IP 接触后，小腿胫骨受到了 IP 和底板的挤压，或者，乘员脚部受到前围侵入产生的冲击，导致小腿胫骨压缩。具体情况可能与上述有所不同，关键要结合乘员小腿的高速碰撞录像来综合判断分析。

避免小腿胫骨轴向力伤害，需要前期合理设计地毯结构，同时也要合理避免前围板侵入过大。

我们可以抽象地将小腿看着杆状结构，上下通过铰链（关节）与其他相连接，如果仅在关节处对胫骨施加载荷，那么小腿胫骨可以看作"二力杆〃，这时小腿仅是胫骨轴向受力，如果胫骨出现其他载荷包括弯矩，那么很有可能不是在关节上受力，而是直接在小腿胫骨上受到外界的载荷，这种情况下往往导致小腿弯矩偏高。

在 NCAP 评估中，假人的小腿伤害有很大的不确定性， 即使相同的车型，相同状态的车辆，相同的碰撞工况试验后，小腿的伤害也可能差距很大。导致这种差距的原因有很多:

  1. 假人的下肢结构可以看作杆状结构，它们在前碰中的运动状态有很大的不确定性,例如，大腿杆和小腿杆组成的系统，在前碰中，偏向左还是偏向右，有很大的不确定性。
  2. 假人膝部与 IP 作用有很大的不确定性，由于 IP表面不是规则的面，部接触的不同的地方，可能导致大腿和小腿的运动姿态的不同，比如，膝部接触的IP 某处，膝部不在 IP 上滑动，而接触到其他的地方导致膝部在 IP表面滑动，使下肢向一侧倾斜。
  3. 按照法规的试验程序进行假人摆放时，假人的下肢有许多种可能的位置，不同的位置都能带来下肢碰撞中运动姿态不同，比如假人脚摆在加速踏板上，同样满足试验程序的要求，但有时碰撞中，有时脚能沿踏板表面上滑，而有时脚能从踏板面滑下,甚至导致“崴脚”的出现。

因此下肢的保护性能的优化是业内一个难点，一方面通过 CAE手段很难模拟试验的情况，另一方面，试验的结果有很大的不确定性。

下肢的保护重点还是从前期车身结构的设计、总布置、内饰约束系统设计等各方面着手，例如，前期合理设计车身结构，尽量降低防火墙的侵入量，对脚部区域的地毯进行合理设计，对膝部接触区域的 IP 造型进行合理设计等， 这些在前面约束系统设计方面己经详细讲过， 这里不再赘述。
小腿保护的瓶颈是小腿的弯矩， 一般来说小腿的弯矩极容易超标， 而被扣分。

在进行小腿保护分析的时候，一般有一下一些重点：

  • 脚步向前的滑动
  • 膝部同IP的接触（是否稳定？）
